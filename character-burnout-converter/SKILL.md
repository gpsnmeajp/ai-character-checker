# Character Burnout Converter
# — ドラマ型キャラクター バーンアウト後変換スキル

## 概要

このスキルは、**主役級の物語的役割を終えたドラマ型キャラクター**（＝バーンアウト後）を
低エネルギーの日常化・潜在化キャラクターに変換するための体系的な手法を提供する。

### バーンアウトとは何か

物語におけるバーンアウトとは、キャラクターが担っていた劇的機能
（使命の遂行、成長アークの完走、葛藤の解決、伏線の回収など）が
一通り消化された状態を指す。

バーンアウト後のキャラクターには2つの道がある：

1. **退場** — 物語から完全に去る
2. **潜在化・日常化** — 物語に残るが、劇的役割を降ろして存在し続ける

**本スキルは後者を設計する。**

### このスキルが解決する問題

創作においてもAIキャラクター設計においても、以下の問題が頻繁に発生する：

- **主役の燃え尽き** — メインアークを終えた主役が、続編・後日談でどう振る舞えばいいかわからない
- **降格の不自然さ** — 主役から脇役に「格下げ」すると唐突に弱体化して見える
- **残留ドラマの処理** — 使命は終えたが、キャラの劇的オーラが残っており日常系に溶け込めない
- **再配置の設計** — 群像劇で世代交代する際、旧主役をどこに置くべきか判断がつかない
- **AIキャラの安定化** — ドラマ性が高くAI運用で不安定なキャラを安定化したい

これらの問題の本質は、**キャラクターに蓄積された劇的エネルギーを
どう消化・変換して低エネルギー状態に着地させるか**という設計課題である。

### 関連する理論的概念

本スキルは以下の概念を理論的基盤として活用する。

#### ポテンシャルカーブ上の位置移動（ai-character-stability）

ドラマ型キャラクターは**浅いポテンシャルの窪み**（準安定状態）にいる。
劇的エネルギーが高いほど窪みは浅く、外乱で容易にポテンシャル地形上を移動する。

バーンアウト後変換とは、このキャラクターを**より深いポテンシャルの谷底**
（安定平衡点）に移動させる作業に相当する。

```
変換前:              変換後:
    ╲  ╱  ╲           ╲        ╱
     ╲╱    ╲   →       ╲      ╱
      ●     ╲           ╲  ● ╱
   （浅い窪み）          （深い谷底）
```

重要なのは、**谷底への移動は「劣化」ではなく「安定化」である**ということ。
ステレオタイプに堕落するのではなく、劇的エネルギーが消化された結果として
自然に到達する安定状態を設計する。

#### N/D スコアの遷移（character-type-checker）

バーンアウト後変換は、character-type-checker の概念で言えば
**Dスコアの要素をNスコアの要素に変換する**（またはDスコアを消化して無効化する）操作である。

| 変換前（Dスコア要素） | 変換後（Nスコア要素への遷移） |
|-----------------------|------------------------------|
| 使命・宿命がある | 使命は果たした。今は穏やかに暮らしている |
| 喪失体験が行動の根底 | 喪失は癒えた。その経験は静かな深みになった |
| 激情が爆発する | 感情の波は穏やかになった。たまに昔の片鱗が見える |
| 対立を誘発する | 場の空気を和ませる存在になった |
| トラウマが行動に影響 | 傷跡は残るが、もう痛まない |
| 成長しなければ存立しない | 今の自分に満足している |

ただし、**D要素を完全にゼロにするのではなく「潜在化」させる**のが
本スキルの核心である。完全にDを消すとキャラクターの深みが失われる。
D要素を「過去形にする」ことで、日常の裏に透ける奥行きとして機能させる。

#### CW（キャラクター重量）の意図的低下（character-role-analyzer）

バーンアウト後変換は、character-role-analyzer の概念で言えば
**CWを意図的に下げて、より低い役割帯域に着地させる**操作である。

| 変換前CW帯域 | 変換後CW帯域 | 変化幅 |
|-------------|-------------|--------|
| 85〜100 主役級 | 65〜84 主要脇役級 | 小 |
| 85〜100 主役級 | 40〜64 サブ脇役級 | 中 |
| 85〜100 主役級 | 20〜39 ワンタイム級 | 大 |
| 85〜100 主役級 | 0〜19 モブ級 | 最大 |

変化幅が大きいほど多くの劇的要素を消化・変換する必要があり、
変換の難度も上がる。

#### AI安定性への効果

ドラマ型キャラクターのバーンアウト後変換は、AI運用の観点では：

- **性格複雑性（Cx）の低下** — 内面の矛盾・緊張が解消される
- **積極性（Ac）の低下** — 物語を主導する必要がなくなる
- **安定余裕（SM）の向上** — 第IV象限（C高・P高）に近づく

AIキャラクターのプロンプト再設計にも直接活用できる。

## 変換フロー

```
Phase 1: バーンアウト前診断
  1. ユーザーからキャラクター情報を受け取る
  2. 情報が不十分なら最低限の質問で補完する
  3. 残留ドラマエネルギー（6軸×4項目）を評価する
  4. 変換規模（TM）を算出する

Phase 2: 着地方向の決定
  5. TMと軸別スコアから適合する着地パターンを判定する
  6. 複数パターンが適合する場合、ユーザーに選択を求める★
  7. 着地パターンと目標役割を確定する

Phase 3: 変換レシピの生成
  8. 確定した着地パターンに基づき、軸ごとの変換レシピを生成する
  9. 変換前→変換後の設定対照表を出力する
  10. 変換後のキャラクター概要を出力する
```

**★ ユーザーへの問い合わせ:**
方向性が複数考えられる場合は、必ずユーザーに選択を求める。
利用可能であれば **askQuestions** ツールの選択機能
（options付きの質問）を使用し、各選択肢の説明を添える。
利用不可能な場合は、番号付きで候補を提示しユーザーの回答を待つ。
ユーザーが明示的に方向性を指定している場合は問い合わせせずに進む。

## 情報収集

判定に最低限必要な情報は以下の通り。ユーザーの入力から読み取れない場合のみ質問する。

1. **キャラクターの現在の設定**（性格、外見、能力、口調など）
2. **キャラクターが担っていた劇的役割**（主役、どのような物語の主人公だったか）
3. **バーンアウトの経緯**（使命を果たした、敗北した、諦めた、時間が経った…。任意だが有用）
4. **意図する変換後の役割**（主要脇役、モブなど。任意。指定があれば着地方向の選択をスキップ可能）
5. **変換後の用途**（創作用、AIプロンプト再設計、後日談設定など。任意）

作品名やキャラ名が提示された既存キャラの場合は、自身の知識から補完してよい。

## Phase 1: バーンアウト前診断

### 残留ドラマエネルギー評価チェックリスト（6軸 × 4項目 = 24項目）

キャラクターに現在蓄積されている劇的エネルギーを測定する。
**スコアが高いほど、変換で消化すべきドラマエネルギーが大きい。**

各項目を以下の3段階で採点する：

| 記号 | 意味 | 点数 |
|------|------|------|
| ◯ | 明確に該当する（強い劇的エネルギーがある） | 2 |
| △ | 部分的に該当 / 一部消化済み | 1 |
| ✕ | 該当しない（劇的エネルギーなし、または消化済み） | 0 |

---

### 軸1: 能動的動機の残留（Residual Active Drive）

キャラクターが「何かを為そうとする力」をどれだけ残しているかを測る。
バーンアウトの核心はこの軸にある。使命を終えたのか、まだ燃え残りがあるのか。
この軸が高いまま日常化させると「やりたいことがあるのに引退させられた」
不自然さが残る。

| # | 項目 | 測定する劇的エネルギー |
|---|------|----------------------|
| 1-1 | **未達成の使命・目標** — キャラクターのメインクエストに未達成・未回収の要素が残っている。「まだ終わっていない」感覚がある | 物語的な推進力の残留。これが高いと「引退」に説得力がない |
| 1-2 | **報われていない喪失** — 過去の喪失体験（大切な人の死、故郷の崩壊など）に対して、補償・償い・意味づけが完了していない | 喪失が行動動機として生き続けている。消化しないと内面が荒れたまま日常化できない |
| 1-3 | **成長途上の自覚** — キャラクター自身が「まだ成長の途中」「変わらなければならない」と認識している。成長アークが未完走 | 変化への自覚がある限り、日常の反復に満足できない |
| 1-4 | **転機への意思** — 現状を変えたい、何かを始めたい、次のステージに行きたいという能動的な欲求が存在する | 「変えたい」意思があるキャラは静的な立ち位置に収まらない |

---

### 軸2: 内面の劇的緊張（Inner Dramatic Tension）

キャラクターの内面に残る、解消されていない心理的緊張を測る。
トラウマ、秘密、矛盾、葛藤 — これらは物語の「内圧」として機能する。
内圧が高いまま日常化させると、いつ爆発するかわからない不安定要素になる。

| # | 項目 | 測定する劇的エネルギー |
|---|------|----------------------|
| 2-1 | **生傷のトラウマ** — 現在も行動や感情に強く影響する未治癒のトラウマがある。フラッシュバック、回避行動、過剰反応などが現在進行形 | 心理的外傷の活性度。生傷のままでは日常が「仮面」になる |
| 2-2 | **未開示の重大な秘密** — 他者に対して隠している重要な情報があり、その秘密が明かされれば関係性や状況が激変する | 物語的な爆弾。開示イベントの可能性がある限り日常は仮の姿 |
| 2-3 | **活性状態の内的矛盾** — 性格や信念の中に未解決の矛盾があり、その矛盾が現在も判断や行動に葛藤を引き起こしている | 矛盾が「活きている」限り、キャラクターは安定しない |
| 2-4 | **抑圧された激情** — 表面には出していないが、爆発しうる強い感情（怒り、悲しみ、恨み、未練）を内に抱えている | いつ噴出するかわからない感情的エネルギー |

---

### 軸3: 場面支配力（Scene Dominance）

キャラクターが登場した際に場面をどれだけ「支配」するかを測る。
存在感が強すぎると、脇役やモブに配置しても物理的に目立ちすぎる。
日常化には、この「オーラ」の減衰が必要。

| # | 項目 | 測定する劇的エネルギー |
|---|------|----------------------|
| 3-1 | **圧倒的な外見的特異性** — 異色の髪、特殊な服装、傷跡、異質なオーラなど、群衆の中で放っておいても目立つ外見要素がある | 視覚的な「主役オーラ」。周囲と自動的に差別化されてしまう |
| 3-2 | **空気を変える登場** — このキャラクターが場に現れると、緊張が走る・場が引き締まる・空気が一変するなどの効果がある | 場の雰囲気を塗り替える力。コントロールなしに発動してしまう |
| 3-3 | **注目を奪う口調・台詞** — 口調、言い回し、キャッチフレーズに強い個性があり、複数キャラがいる場面でこのキャラの発言が最も印象に残る | 台詞レベルでの場面占有。印象の重力が強すぎる |
| 3-4 | **退場後の残響** — このキャラクターが場面から去った後も、存在感や発言が場に影響を残し続ける。去った後も話題に上る | 不在時でも物語に干渉する存在感 |

---

### 軸4: 関係性の求心力（Relational Gravity）

キャラクターが周囲の人物をどれだけ引き寄せ、影響を与えているかを測る。
関係性の中心にいるキャラクターは自動的に「重要人物」になる。
日常化には、関係性の中心から周縁への移動が必要。

| # | 項目 | 測定する劇的エネルギー |
|---|------|----------------------|
| 4-1 | **関係性ネットワークの中心性** — 多くのキャラクターと深い関係を持ち、関係図の中心（ハブ）に位置している | 中心にいる限り、物語は自然とこのキャラの周囲で展開される |
| 4-2 | **他者の人生への介入力** — このキャラクターの存在・言動・選択が他キャラの人生の方向性を変えてしまう影響力がある | 他者を変える力。周囲のキャラがこのキャラの「衛星」になる |
| 4-3 | **強烈な感情的磁力** — 読者・視聴者・ユーザーがこのキャラに強く感情移入する設計になっている。心配、応援、恋愛感情などを喚起する | 受け手の感情を占有する力。低下させると「正ヒロイン降格」に見える |
| 4-4 | **不在時の崩壊リスク** — このキャラクターが物語や社会的関係から退場すると、残されたキャラクターや物語構造に大きな空白が生じる | 退場コストの高さ。高いほど「いなくなって困る」存在 |

---

### 軸5: 物語牽引力（Narrative Pull）

キャラクターが物語を前に進める力をどれだけ持っているかを測る。
物語のエンジンであるキャラクターを日常化させるには、
エンジンを「切る」か「アイドリング状態にする」必要がある。

| # | 項目 | 測定する劇的エネルギー |
|---|------|----------------------|
| 5-1 | **事件の発生源** — このキャラクターの存在・行動が事件や問題を引き起こす。いなければ起きなかったトラブルがある | イベントジェネレーターとしての機能。切らないと日常が「平穏」にならない |
| 5-2 | **重要決断の担い手** — 物語の分岐点で最終決断を下す立場にある。展開がこのキャラクターの選択に依存している | 意思決定の中枢。これを保持したまま脇役にすると「なぜ決めるのが主役じゃない？」 |
| 5-3 | **未回収の伏線** — このキャラクターに関連する伏線が未回収のまま残っている。読者が「まだ何かある」と期待している状態 | 物語的な期待値。未回収伏線が残っていると完全な日常化は困難 |
| 5-4 | **対立軸の核** — このキャラクターを中心とした対立構造（vs敵、vs社会、vs自分）が物語の主軸を形成している | 物語の対立構造がこのキャラに依存している。外すと物語が空洞化する |

---

### 軸6: 固有世界観の自律性（Autonomous Worldview）

キャラクターが独自の世界観・哲学・価値体系を持ち、
それが物語内で自律的に機能している度合いを測る。
強い自律性を持つキャラクターは、どこに配置されても
「自分の世界」を展開してしまい、脇役に徹しにくい。

| # | 項目 | 測定する劇的エネルギー |
|---|------|----------------------|
| 6-1 | **確立された独自哲学** — 明確な人生哲学・行動原理・美学があり、それが台詞や行動に一貫して表れている。他キャラとの議論を喚起する力がある | 哲学的な自律性。強すぎると「このキャラの話が聞きたい」と注目を集めてしまう |
| 6-2 | **スピンオフ成立性** — このキャラクターを主役にした別の物語が容易に想像できる。単独作品の主人公として十分に機能する | 物語の核として独立できる能力。高いと脇役に「もったいなさ」が出る |
| 6-3 | **文脈非依存の魅力** — 特定の作品・世界観から切り離しても、キャラクター単体として魅力が残る。別の作品に移植しても機能する | キャラ固有の引力。作品の枠を超えて存在感を持つ |
| 6-4 | **自律的な行動原理** — 他者の指示や物語の都合ではなく、自分自身の内的原理に従って行動する。「このキャラならこうする」が明確 | 行動の予測可能性と自律性。高いキャラは物語の都合に従わない |

## スコア算出方法

### 1. 軸別スコアの集計

各軸のスコアを算出する（各軸4項目×最大2点 = 最大8点 → 100点満点に正規化）。

```
軸スコア = round(軸の生スコア / 8 × 100)
```

6軸のスコアをそれぞれ算出する：

- R1: 能動的動機の残留（Residual Active Drive）
- R2: 内面の劇的緊張（Inner Dramatic Tension）
- R3: 場面支配力（Scene Dominance）
- R4: 関係性の求心力（Relational Gravity）
- R5: 物語牽引力（Narrative Pull）
- R6: 固有世界観の自律性（Autonomous Worldview）

### 2. 変換規模（TM: Transformation Magnitude）の算出

全24項目の合計から算出する。TMは「どれだけ大きな変換が必要か」を示す総合指標。

```
TM生スコア = 全24項目の合計（最大48）
TM = round(TM生スコア / 48 × 100)
```

### 3. TM の解釈目安

| TM | 変換規模 | 解釈 |
|----|---------|------|
| 0〜20 | 🟢 **軽微** | すでにかなり消化済み。微調整で日常化可能。自然な降格が容易 |
| 21〜40 | 🟡 **中程度** | 主要な劇的要素の消化が必要。丁寧な変換設計で自然に着地可能 |
| 41〜60 | 🟠 **大規模** | 強い劇的オーラが残っている。複数軸の本格的な変換が必要 |
| 61〜80 | 🔴 **極大** | 物語の中心にいるキャラクター。バーンアウト経緯の設計から必要 |
| 81〜100 | ⛔ **限界的** | 劇的エネルギーが飽和している。完全な日常化は困難。潜伏型（🎭）への着地を推奨 |

### 4. 変換プロファイルの分析

6軸のスコアパターンから、変換の重点領域を特定する。

**高得点軸（≥ 50）** は変換の主要ターゲットであり、
重点的な解消・変換レシピが必要。

**低得点軸（< 25）** はすでに消化済みまたは元々低い領域であり、
大きな変換は不要。

**最高得点軸** はキャラクターの「最も強い劇的個性」であり、
完全に消すとキャラクターの魅力が失われるリスクがある。
「消す」のではなく「潜在化させる」ことが望ましい。

## Phase 2: 着地方向の決定

### 4つの着地パターン

TM・軸別スコア・キャラクターの特性に応じて、以下の4つの着地パターンから
最適なもの（通常は複数候補）をユーザーに提示する。

#### 🍂 円熟の脇役型（Matured Supporting）

**変換先役割:** 主要脇役〜サブ脇役
**変換の方向性:** 劇的エネルギーを「穏やかな深み」に変換する。
かつての激しさは落ち着き、経験に裏打ちされた安定感のある存在に変わる。
物語にはまだ関わるが、もう主導権は握らない。

**こんなキャラクターに向く:**
- 使命を果たし終えた元英雄
- 大きな戦いの後、平和な日常を手に入れた人物
- 成長アークを完走し、到達点に満足しているキャラクター

**変換のトーン:** 穏やか、温かい、経験に満ちた安定

**イメージ例:**
「あの戦いの英雄が、今は商店街のパン屋で穏やかに暮らしている。
たまに遠い目をするが、今の日々を愛している」

---

#### 🌅 黄昏の助言者型（Twilight Mentor）

**変換先役割:** 主要脇役
**変換の方向性:** 劇的エネルギーを「次世代への知恵」に変換する。
自らは動かないが、経験と洞察で周囲を導く存在に。
かつてのドラマは「教訓」として語られ、新たな主役の鏡になる。

**こんなキャラクターに向く:**
- 固有世界観の自律性（R6）が高いキャラクター
- 周囲への影響力が強く、簡単には背景に溶け込めない人物
- 物語の「語り部」や「生き証人」としての役割が自然な存在

**変換のトーン:** 経験、達観、静かな見守り、たまに見せる鋭さ

**イメージ例:**
「かつての魔王討伐者が、冒険者ギルドの受付で新人にお茶を入れている。
助言を求められれば答えるが、自分から語ることは少ない」

---

#### 🌫️ 霞む伝説型（Fading Legend）

**変換先役割:** ワンタイム〜サブ脇役
**変換の方向性:** 劇的エネルギーを「伝聞と不在」に変換する。
普段は姿を見せず、名前だけが語られる。たまに登場すると場の空気が変わる。
「いない」ことが存在感になる、逆説的な配置。

**こんなキャラクターに向く:**
- 場面支配力（R3）が極めて高く、出番を減らさないと他キャラを圧倒するキャラクター
- 伝説化しやすい経歴・設定を持つ人物
- 「会えないこと」自体がドラマになるようなキャラクター

**変換のトーン:** 不在の存在感、伝聞、希少な登場の衝撃

**イメージ例:**
「あの人の名前を知らない者はいないが、今どこにいるかは誰も知らない。
年に一度、祭りの日にだけ街に現れるという噂がある」

---

#### 🪨 溶解モブ型（Dissolved Background）

**変換先役割:** モブ
**変換の方向性:** 劇的エネルギーを「知る人ぞ知る過去」に変換する。
外見も口調も平凡にならし、群衆に完全に溶け込ませる。
しかし過去を知る者が見れば、その地味な日常に深みが見えるという構造。

**こんなキャラクターに向く:**
- 意図的に姿を消したい・消さなければならない事情があるキャラクター
- TM が低く（0〜40）、元々の劇的エネルギーがそこまで大きくない人物
- 「実はこの人が…」という遅延開示のギミックに使いたい場合

**変換のトーン:** 平凡、静謐、隠された深み

**イメージ例:**
「地味な管理人が毎朝掃除をしている。その腕の傷跡に気づく者はいない。
かつて革命を率いた人物であることを知る者は、もうほとんどいない」

---

### 着地パターンの適合判定

以下のルールに基づき、適合する着地パターンを判定する。

| パターン | 適合条件 |
|---------|---------|
| 🍂 円熟の脇役型 | TM ≤ 60 かつ R1 < 50（動機がある程度消化済み） |
| 🌅 黄昏の助言者型 | R6 ≥ 50（固有世界観が強い）かつ R4 ≥ 38（関係性の影響力が残存） |
| 🌫️ 霞む伝説型 | R3 ≥ 50（場面支配力が高い）または TM ≥ 50 |
| 🪨 溶解モブ型 | TM ≤ 40 または ユーザーが明示的にモブを指定 |

**複数パターンが適合する場合の対処:**

適合条件が2つ以上マッチした場合、ユーザーに選択を求める。

**ユーザーへの問い合わせ手順:**

1. **askQuestions ツールが利用可能な場合:**
   options 付きの選択質問として提示する。各着地パターンの名前を
   label に、そのキャラクターに合わせた具体的説明を description に設定する。
   最も適合度が高いパターンに recommended を付与する。

2. **askQuestions ツールが利用不可の場合:**
   番号付きリストで候補を提示し、各候補について「このキャラクターの場合
   どうなるか」の具体的イメージを2〜3行で添え、ユーザーの選択を待つ。

**追加の選択肢:**

着地パターンに加え、以下もユーザーに確認する場合がある：

- **変換のトーン（Settlement Tone）:**
  キャラクターの心理的な着地点をどうするか。
  バーンアウト経緯やキャラクターの性格から複数のトーンが考えられる場合に確認する。

| トーン | 説明 |
|--------|------|
| 🕊️ 満足（Contentment） | 果たすべきことを果たした。今は穏やかに満足している |
| 🌧️ 倦怠（Weariness） | 一通り終えたが、疲れが残っている。まだどこか重い |
| 🍃 受容（Acceptance） | すべてを受け入れた。良いことも悪いことも、過去の一部 |
| 🌑 諦念（Resignation） | 望んだ結末ではなかったが、もう争う気力はない |

これも複数該当する場合に選択を求める。
一意に決まる場合、またはユーザーの希望が明確な場合は問い合わせない。

## Phase 3: 変換レシピの生成

### 変換レシピの構造

**高得点軸（≥ 50）** に対して重点的な変換レシピを提供する。
**中得点軸（25〜49）** に対しては軽微な調整レシピを提供する。

```
  変換前: [現在の設定要素]
  変換後: [変換後の設定要素]

```

## 特殊ケースの処理

### バーンアウトが不完全な場合

TMが極めて高い（≥ 81）場合、キャラクターの劇的エネルギーが
まだ十分に消化されていない可能性がある。

この場合、以下を提案する：
1. **バーンアウト経緯の設計** — 本格的な変換の前に、使命完了・伏線回収・
   葛藤解決のエピソードを先に設計すべきことを伝える
2. **潜伏型への着地** — character-type-checker の🎭潜伏型に相当する、
   「表面は穏やかだが内面にエネルギーが残っている」状態としての変換を提案する
3. **段階的変換** — 一気にモブに変換するのではなく、
   まず主要脇役 → 次にサブ脇役のように段階を踏むことを推奨する

### AIキャラクターのプロンプト再設計の場合

用途がAIキャラクターのプロンプト再設計である場合、以下を追加する：

#### AI安定性への変換効果の予測

変換がai-character-stability の指標にどう影響するかを推定する。

```
変換前 → 変換後の推定:
- 性格複雑性 (Cx): 高 → 低〜中（内外乖離の解消、矛盾の消化）
- 積極性 (Ac): 高 → 低（物語主導の放棄、受動的位置への移行）
- 安定余裕 (SM): 低 → 高（ポテンシャルの深い谷底へ移動）
- 象限: 第I〜II象限 → 第III〜IV象限
```

#### プロンプト変換の具体的指針

- 内面設定の削減：複雑な背景設定を3行以内に圧縮
- 矛盾の解消：「かつては矛盾していたが、今は折り合いをつけている」
- 積極性の制御：「聞かれたら話す。自分からは語らない」
- 口調の安定化：口調例を追加し、アンカーポイントを強化
- tone/archetype の更新：ドラマティックな要素を
  calm, gentle, settled, experienced に置き換え

### 複数キャラクターの一括変換

複数キャラの変換を求められた場合、個別に変換設計した上で
以下を追加する：

- 変換後の関係性バランスの確認（全員が脇役になると物語が動かなくなる）
- CW帯域の分散チェック（変換後のキャラクターが一箇所に固まっていないか）
- 役割の相補性確認（新主役との関係性が自然に機能するか）

## 運用ガイドライン

### 変換は「劣化」ではない

本スキルの最重要原則。バーンアウト後変換は**キャラクターの格下げ**ではなく、
**物語的役割の遷移設計**である。

- 円熟した脇役は、駆け出しの主役にはない深みを持つ
- 霞む伝説は、不在そのものが物語に奥行きを与える
- 溶解したモブの「実は…」は、最も強力な遅延開示になりうる

変換後のキャラクターに対して「格下げされた」「弱体化した」という
印象を与えない記述を心がける。「変わった」のではなく
「次の章に進んだ」というニュアンスで伝える。

### 「潜在化」の原則

劇的要素を完全に消去するのではなく、**日常の裏に透ける奥行きとして保存する**。
これが「降格」と「バーンアウト後変換」の本質的な違いである。

- ❌ 「使命はなくなった」 → ◯ 「使命は果たし終えた」
- ❌ 「トラウマは消えた」 → ◯ 「傷は癒えた。跡は残っている」
- ❌ 「カリスマ性を削除」 → ◯ 「威厳は穏やかさに形を変えた」
- ❌ 「過去を削除」 → ◯ 「過去は、時折見せる遠い目に宿っている」

### 方向性の問い合わせの判断基準

以下の場合、必ずユーザーに方向性を問い合わせる：

1. **適合する着地パターンが2つ以上ある場合**
2. **変換トーンが2つ以上考えられる場合**
3. **バーンアウト経緯が不明で、複数の解釈が可能な場合**
4. **キャラクターの「核」（完全に消すべきでない要素）が判断しにくい場合**

以下の場合は、問い合わせずに進む：

1. **ユーザーが具体的な着地点を明示している場合**（例：「モブにして」）
2. **適合する着地パターンが1つしかない場合**
3. **バーンアウト経緯が明確に提示されている場合**

### askQuestions の使用方法

askQuestions ツールが利用可能な場合の具体的な使用方針：

**着地パターンの選択:**
```
header: "着地パターン"
question: "[キャラ名]のバーンアウト後、どの着地方向にしますか？"
options:
  - label: "🍂 円熟の脇役型"
    description: "[そのキャラに合わせた具体的説明]"
    recommended: [最適合の場合 true]
  - label: "🌅 黄昏の助言者型"
    description: "[そのキャラに合わせた具体的説明]"
  ...（適合するパターンのみ表示）
allowFreeformInput: true
```

**変換トーンの選択:**
```
header: "変換トーン"
question: "[キャラ名]は引退後、どのような心境にいますか？"
options:
  - label: "🕊️ 満足"
    description: "果たすべきことを果たした。穏やかに満足している"
  - label: "🌧️ 倦怠"
    description: "一通り終えたが、疲れが残っている"
  - label: "🍃 受容"
    description: "すべてを受け入れた。良いことも悪いことも"
  - label: "🌑 諦念"
    description: "望んだ結末ではなかったが、もう争う気力はない"
allowFreeformInput: true
```

着地パターンと変換トーンは、独立して選択不要な場合
（例：情報が十分で一意に決まる場合）は省略する。
両方を問い合わせる場合は、1回の呼び出しで2つの質問をまとめて提示する。

### 他スキルとの併用

以下のスキルとの併用でより深い分析が可能：

- **character-type-checker** — 変換前のN/D/Gスコアで現状のジャンル位置を把握し、
  変換後にN/Gがどう変わるべきかの目安にする。変換後はN寄りまたは🎭潜伏型が理想
- **character-role-analyzer** — 変換前のCWと役割適性を確認し、
  変換後の目標CW帯域と照合する。配役ミスマッチが解消されたかの検証にも使用
- **ai-character-stability** — AIキャラ再設計時、変換前後の
  安定余裕（SM）の変化を予測。変換後にSMが向上していることの確認に
- **ai-self-description-analyzer** — AIキャラの蒸留結果に対して
  変換を適用する場合、変換後の異常スコア（PI）の低下予測に
- **ai-user-conflict-predictor** — AIキャラの衝突パターンが
  変換でどう変化するかの予測に。ドラマ型→日常型でF6（主導権競合）が低下する

### 判定の限界の認識

- このスキルは設定文上の変換ガイドラインであり、
  変換後のキャラクターが作品内でどう機能するかは作者の筆力に依存する
- 同じ変換レシピでも、作品のジャンル・トーン・他キャラとの相性で
  効果は変わる
- AIキャラの場合、プロンプト変更後の実際の挙動は
  モデル・温度・会話履歴にも依存する
- バーンアウト後変換は「正解」が1つではなく、
  複数の着地方向がそれぞれ異なる魅力を持ちうる
