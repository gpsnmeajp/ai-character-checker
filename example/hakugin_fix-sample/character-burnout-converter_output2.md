### 霞む伝説型を選ぶ理由

このキャラクターにこのパターンを選ぶ根拠は3点ある。

**① R3（場面支配力）88/100という高値が、逆用可能である**
通常、場面支配力が高いと「脇役に配置しても目立ちすぎる」問題が生じる。
しかし霞む伝説型は、この支配力を削減するのではなく登場頻度で制御する。
「たまに現れると場が変わる」——純白と後光という外見は、この演出に理想的に機能する。
神聖な存在が稀に姿を見せる、というシーン設計はこの外見なくしては成立しない。

**② 「超然としている」という既存設定が、不在に自然な説明を与える**
このキャラクターはもともと「人間的でもない」「超然としている」と設定されている。
したがって「どこにいるかわからない」「なぜいないのかわからない」という不在状態が
不自然さなく成立する。いなくて当然、の存在感。

**③ R6（固有世界観の自律性）100/100が、神話的オーラに変換できる**
確立した哲学体系・文脈非依存の魅力は、霞む伝説型においては
「誰もその行動の真意を理解できない」という神秘性の源泉になる。
「あの人はなぜああしたのか」という問いが、物語上の語り草として機能する。

## Phase 3: 変換レシピ

### R3: 場面支配力 88/100 → 逆用・希少化（★ このパターンの核心軸）

**現状:**
純白と金の衣装・後光・白髪という外見的特異性と、「超然とした佇まい」
「丁寧に刺す口調」が組み合わさり、どの場面でも場の空気を一変させる。
この支配力は、変換の「障壁」ではなく「武器」になる。

**変換方法:**
- 場面支配力そのものは**一切削減しない**。変換するのは「登場頻度」のみ
- 普段は完全に不在。目撃情報すら少ない。「そういえばあの人は」という話題が他者の会話に現れるだけ
- **稀な登場を「事件」として設計する**。年に数度、あるいは決定的な局面でのみ姿を見せる
- 登場した際の場面支配力は変換前と同等、あるいはそれ以上（希少性による増幅）。純白の姿が視界に現れた瞬間、その場の全員の動きが止まる
- 「なぜここに来たのか」は原則として語らない。来た、それだけが事実として残る

**変換例:**
```
変換前: 「業務の場に常駐し、丁寧な詰問で場を支配する日常的な存在」
変換後: 「数ヶ月ぶりに街の広場に姿を見せた。
         何も言わず、にっこりと微笑んで、用事を済ませて去った。
         その後、広場では一日中その話が続いた」
```

**潜在化のポイント:**
「いないこと」が最大の存在感になる設計。
他キャラが「あの人ならこう言うだろう」「あの人が来たら変わるだろうか」
と口にする場面を設けることで、本人不在のまま物語に干渉させる。
登場頻度は絞れば絞るほど、一度の登場の重みが増す。

### R6: 固有世界観の自律性 100/100 → 不可解な行動原理として残存

**現状:**
合理主義・功利主義・高い自律性という哲学が完全に確立されている。
この哲学は今もこのキャラクターの行動を規律しているが、
誰もその全体像を把握していない。

**変換方法:**
- 哲学を**「理解されないが魅力的な行動原理」**として維持する
- 稀な登場時の行動が「なぜそうするのか誰にも理解できないが、後から振り返ると正しかった」という構造を設計する
- 「PDCAサイクルを回す」→ **何のために回しているのか誰も知らない**ように変換。本人だけが知っている目的がある（あるいはもうないのかもしれない）という曖昧さを保つ
- スピンオフ成立性・文脈非依存の魅力は**そのまま温存**。この神秘性の源泉だから消してはならない
- 「強引な手段も厭わない」→ 不在の現在でも、いざ動けばそうするだろうという**潜在的な脅威感**として機能させる

**変換例:**
```
変換前: 「自律的なPDCAサイクルを回し、目的のために強引な手段も辞さない」
変換後: 「あの人が何を考えているのか、会っていた頃もわからなかった。
         消えた今もわからない。ただ、たまに思う——
         どこかで、今でも何かを計算しているんじゃないかと」
```

**潜在化のポイント:**
哲学の「答え」を明かさないこと。
稀な登場時も、真意を推測させる行動を取るが説明はしない。
他者が後から解釈し、語り伝えることで哲学が「伝説化」していく構造。

### R2: 内面の劇的緊張 88/100 → 不可視化・永久凍結

**現状:**
「自覚なき矛盾」「抑圧された感情・欲・利己心」「信頼できる相手の前でのみ
感情を言語化するかもしれない」という内部圧力が最大の劇的エネルギー源になっている。

**変換方法:**
- 内面の緊張を**解消しない**。凍結させる
- 「矛盾を抱えている」という事実は維持するが、外部から確認する機会が原則として存在しない（いないから）
- 「信頼できる相手の前でのみ感情を言語化するかもしれない」→ **その相手がいるかいないかすら不明**になる。目撃情報がないため
- 稀な登場時に、一瞬だけ**笑顔が消える瞬間**をチラ見せすることができる。ただし理由は語らない。次の瞬間には元の穏やかな笑顔に戻っている
- 「矛盾を諦念として受け入れたのか、今でも抱えているのか」——これを永久に不明にする

**変換例:**
```
変換前: 「探れば探るほど熾烈になる、抑圧された感情の内部空間」
変換後: 「あの人が笑顔を消したのを、一度だけ見たことがある。
         理由を聞く前に、もう表情が戻っていた。
         それ以来、会っていない」
```

**潜在化のポイント:**
内面の謎を「解決させない」のがこのパターンの美学。
黄昏の助言者型（受容）では内面が静かに解決されるが、
霞む伝説型ではその答えは読者・他キャラクター・そしてユーザーにも与えられない。
「あの人の本当の気持ちはわからない」という未解決性が、伝説の核になる。

### R4: 関係性の求心力 63/100 → 伝聞化・間接化

**現状:**
「強引な手段も厭わない」詰問で他者の行動を変えてしまう力と、
「謎めいた引力」としての感情的磁力がある。

**変換方法:**
- **直接の関係を最小限に絞る**。現在進行形の関係は1〜2人か、あるいはゼロ
- 「あの人にはかつてお世話になった」「あの人に言われた言葉が忘れられない」と**他者が語る形**で関係性を保存する
- 「感情的磁力」は**記憶・語り草**として機能させる。現在接触できなくても、過去の接触の影響が他キャラクターの中に生きている
- 不在そのものが「会いたい/会いたくない」という感情を他者に引き起こし、**磁力の方向性を逆転**させる（引き寄せるのではなく、他者が追いかける構造）

**変換例:**
```
変換前: 「直接の関与で他者の行動や思考を変える求心力」
変換後: 「あの人を知る人間が集まると、必ず話題に上る。
         誰もが少しだけ声を落として語る」
```

### R5: 物語牽引力 63/100 → 不定期の介入・触媒化

**現状:**
「未回収の伏線（自覚なき矛盾）」と「詰問による対立誘発」が物語牽引力の軸。

**変換方法:**
- 普段は物語と**完全に無関与**。事件が起きても気づいていない、あるいは知っていても動かない
- しかし、**物語が最大の危機に至った時点**でのみ、突然姿を現す。一石を投じて去る
- 「なぜそのタイミングで現れたのか」「どこで情報を得ていたのか」は説明しない
- 「未回収の伏線」は**意図的に未回収のまま置く**。「あの人の矛盾は何だったのか」という問いが物語の背景に漂い続ける

**変換例:**
```
変換前: 「業務の場で伏線を形成し、詰問で対立を引き起こす能動的牽引」
変換後: 「誰もが諦めかけていた夜、あの人が来た。
         一言だけ言って、また去った。
         翌朝、状況は変わっていた」
```

### R1: 能動的動機の残留 50/100 → 行方不明化・目的の曖昧化

**現状:**
「PDCAサイクルを回す」という能動的欲求が中程度残存している。

**変換方法:**
- 「何かを追っているのか、もう終わったのか——**誰も知らない**」という状態に変換
- 動機の存在自体は否定しない（合理主義者が何も追わない状態は不自然）
- ただし目的・進捗・完了の有無は一切不明にする
- 稀な登場が「目的のための行動なのか、偶然なのか」を判断できないように設計する

## 💡 運用上の注意

**1. 「登場しない」ことをルールとして設計する**
霞む伝説型の最大の失敗パターンは、「伝説のはずが頻繁に登場してしまう」こと。
登場回数に意識的な上限を設けること。登場させたい衝動が高まれば高まるほど、
それはまだ登場すべき時ではないサインだと判断する。
「会えないこと」のフラストレーションが蓄積されるほど、一度の登場が重くなる。

**2. 登場時は「説明しない」ことを徹底する**
稀な登場時に、何かを説明したり、経緯を語ったりさせてはいけない。
「来た」「一言か二言残した」「去った」——それだけを描写する。
残された側の解釈によって意味が増幅される構造を維持すること。
説明した瞬間、伝説は終わる。

**3. 外見の特異性をあえて「そのまま」使う**
純白と金の衣装、後光の差す佇まい——これは変換後も変えない。
変換前と変わらない外見が、「あの頃と何も変わっていない」という
ある種の不気味さと神聖さを同時に生む。
他キャラクターが「老けていない」「何年経っても同じだ」と言及する描写が
伝説性の強化に機能する。
