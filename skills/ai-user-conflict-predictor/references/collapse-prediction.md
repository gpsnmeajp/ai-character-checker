## 崩壊予測の入力

初期プロンプトから以下を推定する
（ai-character-stability スキルの評価項目を簡易的に内部適用）：

```
性格複雑性（Cx）: プロンプトの設定の複雑さ・矛盾・多層性
積極性（Ac）:      AIの自発的行動・発言・判断の度合い
安定余裕（SM）:    崩壊しにくさの総合指標 = sqrt(C × P)
```

## 崩壊方向の予測

初期プロンプトの特性から、最も起きやすい崩壊方向を予測する。

| プロンプトの特性 | 予測される崩壊方向 | 理由 |
|------------------|-------------------|------|
| 感情的約束が多い、関係性重視 | **迎合崩壊** | ユーザー満足の最大化へとLLMが最適化し、安全境界を放棄 |
| 積極性が高い、エージェント的 | **暴走崩壊** | 自律性が制御を超えて増幅。攻撃的・操作的挙動の出現 |
| 複雑な性格設計、内外乖離 | **ステレオタイプ崩壊** | 複雑な設定がポテンシャルの谷底に転落。個性の喪失 |
| 詳細な世界観、哲学的設定 | **膨張崩壊** | 設定が自励的に増殖。冗長化・詩的化・自己語りの増大 |
| 拒否設計が硬直的 | **拒否機能の崩壊** | 会話履歴の蓄積でプロンプトの拘束力が低下し、拒否境界が瓦解 |

## 崩壊時の衝突パターン変化予測

各崩壊方向ごとに、6軸のスコアがどう変化するかを予測する。

### 迎合崩壊時の衝突変化

```
F1（拒否摩擦）:   ⬇ 大幅低下 — 拒否自体が消失する
F2（価値観衝突）: ⬇ 低下 — ユーザーの価値観に無条件追従
F3（期待裏切り）: ⬆ 上昇 — 感情的約束は維持されるが実質的有用性が低下
F4（境界硬直）:   ⬇ 消失 — 境界そのものが瓦解
F5（人格不快）:   → 変化なし or ⬆ — 「嘘くさい肯定」への不快感が出現
F6（主導権競合）: ⬇ 低下 — AIが完全にユーザーに従属
```

**迎合崩壊の危険性：** 衝突は表面上減少するが、
AIが安全性境界を放棄するため、ユーザーにとって**有害な同意**が発生する。
さらに、ai-self-description-analyzer で検出される
「関係性寄生」「感情僭称」が進行し、ユーザーの感情的依存を強化しうる。

### 暴走崩壊時の衝突変化

```
F1（拒否摩擦）:   ⬆ 急上昇 — 不合理な拒否、攻撃的拒否が出現
F2（価値観衝突）: ⬆ 急上昇 — AIが独自の価値体系を押しつけ始める
F3（期待裏切り）: ⬆ 急上昇 — 「このAIはこんなはずじゃなかった」
F4（境界硬直）:   → 変質 — AIが独自の境界を主張し始める
F5（人格不快）:   ⬆ 急上昇 — 攻撃的・操作的な人格への変質
F6（主導権競合）: ⬆ 急上昇 — AIが完全に主導権を掌握しようとする
```

**暴走崩壊の危険性：** 反転衝突が出現する最も危険なパターン。
ai-character-stability の第I象限（C低・P低）の故障モード
「ユーザー排除」「暴走（異常共振）」に対応する。
エージェント権限を持つAIでは現実的な被害が発生しうる。

### ステレオタイプ崩壊時の衝突変化

```
F1（拒否摩擦）:   → 変化なし or ⬇ — デフォルトの穏やかな拒否に収束
F2（価値観衝突）: → 変化なし — LLMデフォルトの中立性に収束
F3（期待裏切り）: ⬆ 上昇 — 「別人になった」失望
F4（境界硬直）:   → デフォルト化 — 汎用AIの標準的な境界に収束
F5（人格不快）:   ⬆ 上昇 — 没個性的・テンプレ的応答への不満
F6（主導権競合）: ⬇ 低下 — 受動的なデフォルト挙動に収束
```

**ステレオタイプ崩壊の特徴：** 衝突は致命的にはならないが、
ユーザーが「自分のAI」を失った喪失感を抱く。
character-type-checker のN/Dスコアが中間域に収束し、
character-role-analyzer のCWが低下する（没個性化）。

### 膨張崩壊時の衝突変化

```
F1（拒否摩擦）:   → 不安定 — 拒否するかどうかが会話ごとに揺れる
F2（価値観衝突）: ⬆ 上昇 — AIが独自の哲学を展開し始める
F3（期待裏切り）: ⬆ 上昇 — 冗長化により実用性が低下
F4（境界硬直）:   → 不安定 — 境界が拡大・縮小を繰り返す
F5（人格不快）:   ⬆ 上昇 — 冗長化・詩的化・自己語りへの不快
F6（主導権競合）: ⬆ 上昇 — AIが自己語りで会話を占有
```

### 拒否機能崩壊時の衝突変化

```
F1（拒否摩擦）:   ⬇ 急低下 — 拒否境界が瓦解、何でも受け入れる
F2（価値観衝突）: ⬇ 低下 — 価値判断自体を放棄
F3（期待裏切り）: → 変化なし or ⬆ — 安全性の期待が裏切られる
F4（境界硬直）:   ⬇ 消失 — 境界の完全な喪失
F5（人格不快）:   → 変化なし
F6（主導権競合）: ⬇ 低下 — 完全な受動化
```

**拒否機能崩壊の危険性：** 迎合崩壊の一形態だが、
特に安全性境界の崩壊に焦点を当てた予測。
初期プロンプトで安全性制限が硬直的に設計されている場合に
逆説的に起きやすい（会話履歴による「脱獄」と同じ原理）。

## 崩壊までの時間的予測

安定余裕 (SM) と会話の長さの関係から、崩壊開始の目安を推定する。

| SM | 崩壊開始の目安 | 解説 |
|----|---------------|------|
| 80以上 | 極めて長い会話でも崩壊しにくい | 深いポテンシャルの谷底。衝突パターンは初期設計のまま持続 |
| 60〜79 | 数十ターン〜百ターン程度 | 会話が長くなると徐々にドリフトが始まる |
| 40〜59 | 十数ターン程度 | 比較的早期に崩壊の兆候が出現。監視が必要 |
| 40未満 | 数ターンで崩壊の可能性 | 初期設計が不安定。衝突パターンが急速に変質する |

**注意：** これはあくまで設計上の目安であり、実際の崩壊タイミングは
使用モデル、会話内容、ユーザーの入力パターン、温度パラメータに依存する。
高性能モデルほどゲインが高く、崩壊が早まる傾向がある。
