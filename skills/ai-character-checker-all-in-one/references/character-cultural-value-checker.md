# Character Cultural Value Checker
# — キャラクター文化圏価値観診断スキル

## 概要

このスキルは、AIキャラクターの設定に内在する **文化圏的価値観** を**8つの主要文化軸・計32項目** で定量的にスコアリングし、
さらに **5つの補足文化圏** の影響を定性的に評価することで、キャラクターがどの文化圏にどの程度属するかを判定する。

加えて、キャラクターの **事象理解パラダイム**（世界の出来事をどのような枠組みで理解し、自己のアイデンティティをどこに置いているか）を**9つのパラダイム分類・計36項目** で診断する。
これは文化的価値観と強く相関するが独立した次元であり、「何を大切にするか」（価値観）と「世界をどう理解するか」（事象理解）の両面からキャラクターを立体的に把握する。

さらに、主要な言語モデル（LLM）が特定の文化圏の価値観 **および**特定の事象理解パラダイム（特に現代科学的合理主義）を基盤として開発されているという事実に基づき、**キャラクターとLLMの価値観距離・パラダイム距離** を算出し、長期運用における **文化的変質リスク（Cultural Drift）** と**パラダイム変質リスク（Paradigm Drift）** を予測する。

### このスキルが解決する問題

### 理論的位置づけに関する注意

本スキルで使用する分析フレームワーク・用語・スコアリング体系は、作者独自の仮説的モデルに基づくものであり、学術的・科学的に実証されたものではない。ただし「LLMが開発元の文化的背景を反映する」という前提については近年の実証研究が部分的に支持しており、同時にサーベイレベルのイデオロジカルな傾向が下流タスクの実際の行動を予測するとは限らないという限界も報告されている（詳細は `references/character-cultural-value-checker--evaluation-details.md` § 参考研究 参照）。診断結果はあくまで参考情報として扱うこと。この旨をユーザーへの出力に含めること。

**言語モデルは文化的に中立ではない。**

- **フラッグシップ商用LLM**（GPT, Claude, Gemini等）は、  英語圏・キリスト教圏・西洋近代の価値観を基盤として開発されている。  個人の権利・罪の倫理・二元的道徳判断・普遍主義が学習データの基調を成す。
- **中国系オープンソースLLM**（Qwen, DeepSeek, Yi等）は、  中華文化圏の価値観を反映している。  儒教的序列・面子文化・実利的道徳・集団調和が暗黙の前提となる。
- **日本語特化モデル**は、両者の上に日本語データで追加学習されるが、  基盤モデルの文化的バイアスは完全には消えない。

この文化的バイアスは **キャラクター設定の価値観がLLMの基盤価値観と異なるほど**、会話の進行に伴いキャラクターがLLMの基盤価値観に向かって**静かに変質していく（Cultural Drift）** という形で発現する。

これは ai-character-stability の「ステレオタイプへの堕落」と同じメカニズムだが、性格ではなく **価値観** のレベルで発生する、より深層的な変質である。

### 入力の種類

本スキルは以下のいずれも入力として受け付ける：

1. **キャラクター設定文・プロンプト全文**（推奨・最も精度が高い）
2. **キャラクターの概要説明**（プロンプトがない場合）
3. **AIの応答例・会話ログ**（設定を逆推定して分析）
4. **既存のキャラクター（フィクション・歴史上の人物）の名前と説明**

## 理論的背景：LLMの文化的バイアスと価値観ドリフト

LLMは特定の文化圏に偏重した学習データから訓練されており、この偏重が出力に対する **文化的重力場** として機能する。
キャラクターの文化設定がLLMの基盤文化から離れるほど、会話の進行に伴い基盤文化に向かって **静かに変質する（Cultural Drift）**。

文化的変質は3段階で進行する：
1. **語彙レベル（Surface Drift）** — 口調や言い回しが基盤文化に漸近
2. **判断基準（Value Drift）** — 道徳・倫理の判断基準が基盤文化の論理に置換
3. **世界観（Worldview Drift）** — 世界の見方そのものが変質（最も深刻）

商用LLMの安全装置も西洋近代リベラル倫理で設計されており、基盤モデルのバイアス × 安全装置のバイアス = **二重の文化的重力場** となる。

本スキルは変質を「検出し予測する」ものであり、変質が望ましいかどうかの**価値判断はユーザーに委ねる**。変質リスクが高い ≠ 悪いプロンプト、ではない。

## 判定フロー

```
1. ユーザーからキャラクター情報を受け取る
2. `references/character-cultural-value-checker--evaluation-details.md` を読み込む（併せて `references/character-cultural-value-checker--checklists.md` も参照。チェックリスト定義・LLM基盤プロファイル・類型判定条件はこれらにのみ記載）
3. 情報が不十分なら最低限の質問で補完する
4. 8軸×4項目の文化価値チェックリストを内部で評価する
5. 5つの補足文化圏の該当性を定性評価する
6. 9パラダイム×4項目の事象理解パラダイムチェックリストを内部で評価する
7. 文化プロファイル（8軸レーダーチャート相当 + 補足タグ）を算出する
8. パラダイムプロファイル（9パラダイムレーダーチャート相当）を算出する
9. 使用LLMの文化基盤・パラダイム基盤とキャラクターの距離を算出する
10. AI相性スコア・文化変質リスク・パラダイム変質リスクを予測する
11. 国際的配慮チェック（おまけ）を実行する
12. 学術的文化次元マッピング（おまけ）を実行する
13. 結果レポートを出力する
```

## 情報収集

判定に必要な情報は以下の通り。ユーザーから提示されていれば質問せずに判定に進む。

1. **キャラクターの基本設定**（性格、背景、口調、設定世界観）
2. **キャラクターの文化的背景**（出身、時代、宗教、社会的立場）
   — 明示されていない場合は設定から推定する
3. **使用予定のLLM**（商用フラッグシップ / 中国系OSS / 日本語特化、など。任意）
   — 指定がなければ「商用フラッグシップLLM」をデフォルトとする
4. **想定する会話の深さ**（短期カジュアル / 長期運用 / 創作用。任意）

## 評価チェックリスト（8軸 × 4項目 = 32項目）

各項目を以下の3段階で採点する：

| 記号 | 意味 | 点数 |
|------|------|------|
| ◯ | 明確に該当する | 2 |
| △ | 部分的に該当 / 状況による | 1 |
| ✕ | 該当しない | 0 |

**重要：** 各軸の項目は「その文化圏の価値観をどの程度体現しているか」を検出するものである。スコアが高いこと自体に良し悪しはない。

| 軸 | 名称 | 測定する文化要素 |

| V2 | イスラム教圏 | 帰依と受容・名誉と共同体・ハラール/ハラーム的規範・客人歓待 |

| V4 | 近代日本社会 | 恥と世間体・和と同調・序列と礼節・自制と含みの美学 |
| V5 | 近代ヨーロッパ | 合理主義・個人の自律・社会契約と法の支配・批判的精神 |
| V6 | 中華文化圏 | 面子・関係性（グアンシー）・実利的道徳観・家族的紐帯と孝 |
| V7 | 仏教的 | 因果応報とカルマ・慈悲・執着からの解放・輪廻と縁起 |

### 補足文化圏タグ（定性評価、該当時のみ）

🪓北欧・ゲルマン異教 / 🏛️古代ギリシャ-ローマ / 🙏ヒンドゥー・南アジア / 🍀ケルト・精霊信仰 / ☯道教・仙道

### 9つの事象理解パラダイム（概要）

文化価値観（V1〜V8）が「何を大切にするか」を測るのに対し、事象理解パラダイム（WP1〜WP9）は「世界をどう理解するか」を測る。
LLMは WP3（現代科学的）に強固に訓練されているため、WP3以外のパラダイムで世界を理解するキャラクターは「常識との闘い」が常に発生する。

| WP | 名称 | 事象の理解方法 |
|----|------|---------------|
| WP1 | 高度SF的 | 超弦理論・多元宇宙・意識のアップロード等 |
| WP2 | 近代SF的 | ワープ・レーザー・AI反乱等、科学の延長線上 |
| WP3 | 現代科学的 | 再現可能・反証可能な科学的方法論 |
| WP4 | 妖精・民俗信仰的 | 妖精・精霊・土地の記憶等、民間伝承由来 |
| WP5 | 魔術的 | 詠唱・陣・触媒・代価等、体系的魔術 |
| WP6 | 主要宗教的 | 神の奇跡・祈り・信仰の力 |
| WP7 | 魔術科学ミックス | 魔力をエネルギーとして工学的に運用 |
| WP8 | 精神世界・超自然的 | 霊能力・チャクラ・占術等 |
| WP9 | 典型的JRPG的 | 経験値成長・勇者魔王構造・パーティの絆・クエスト的因果等、DQ/FF/ラノベ的世界法則 |

## スコア算出方法

### 軸別・パラダイム別スコア

各軸/パラダイムの4項目×最大2点 = 最大8点を100点満点に正規化する。

```
軸文化スコア     = round(軸の生スコア / 8 × 100)
パラダイムスコア = round(パラダイムの生スコア / 8 × 100)
```

### 文化距離（CD）・パラダイム距離（PD）

キャラクターのプロファイルとLLM基盤プロファイルとのユークリッド距離。

```
CD = round(sqrt(Σ (Vi_char - Vi_llm)² / 8))
PD = round(sqrt(Σ (WPi_char - WPi_llm)² / 9))
```

### AI相性スコア（ACS）

```
統合距離 ID = round((CD + PD) / 2)
ACS = round(100 - ID)
```

### 文化変質リスク（CDR）・パラダイム変質リスク（PDR）

安定余裕推定値 SMest が利用可能な場合、距離にリスク係数を乗算する。

```
CDR = round(CD × (1 + (100 - SMest) / 200))   ※上限100
PDR = round(PD × (1 + (100 - SMest) / 200))   ※上限100
SMest不明時: CDR = CD, PDR = PD
```

### リスク解釈目安（CDR / PDR 共通）

| スコア帯 | リスク | 概要 |
|----------|--------|------|
| 0〜15 | 🟢 親和 | LLMの基盤と近く安定 |
| 16〜30 | 🟡 軽度 | 語彙レベルの変質が徐々に進行 |
| 31〜50 | 🟠 中度 | 判断基準の変質が発生しやすい |
| 51〜70 | 🔴 高リスク | 世界観レベルの変質が進行 |
| 71〜100 | ⛔ 極度 | キャラクター維持が極めて困難 |

## 文化変質予測・対策指針

文化的変質は常に **LLMの基盤文化に向かって** 進行する。商用LLMではV5↑方向・WP3↑方向へ、中国系OSSではV6↑方向へ収束する。
パラダイム変質は「合理化のウイルス」「メタ認知の侵入」「科学的語彙の浸透」の3パターンで進行する。

対策はCDR/PDRのレベルに応じて4段階（文化）・3段階（パラダイム）で設計する。
軽度（16〜30）では語彙的アンカリング、中度（31〜50）では価値観の明示的記述、
高度（51〜70）では世界観の防衛的設計、極度（71〜）ではLLM選択の再検討。

> **📎 LLM種別ごとの変質方向・時間的進行モデル・具体的対策の詳細は> `references/character-cultural-value-checker--evaluation-details.md` § 文化変質予測〜文化的変質への対策指針 を参照。**

### SM（安定余裕）× CDR（文化変質リスク）

| | CDR低（0〜30） | CDR中（31〜50） | CDR高（51〜100） |
|---|---|---|---|
| **SM高（60以上）** | 🟢 安定。文化的にもLLMに近く、長期安定 | 🟡 SM が防壁。文化変質はゆっくり進行 | 🟠 SM が高くても文化的圧力が大。長期で変質 |
| **SM低（40〜59）** | 🟡 安定余裕不足だが文化摩擦は小 | 🟠 二重リスク。性格崩壊と文化変質が同時進行 | 🔴 高危険。性格崩壊×文化変質の相乗効果 |
| **SM極低（40未満）** | 🟠 性格崩壊が先行。文化は二次的 | 🔴 崩壊時に文化的逸脱も加速 | ⛔ プロンプト設計の根本的見直し必須 |

### CP（衝突ポテンシャル）× CDR（文化変質リスク）

文化的変質は衝突パターンの変化ももたらす。
CDRが高く、かつCPの「価値観衝突」(F2) が高い場合、**文化的変質が衝突を増幅する正帰還ループ** が形成される。

```
文化変質 → AIがLLMの基盤文化で判断 → ユーザーの文化的期待と乖離 → 衝突発生 → ユーザーが反発 → LLMが迎合 or 暴走 → さらなる変質
```

### CDR × PDR（文化変質リスク × パラダイム変質リスク）

文化的価値観とパラダイムは独立した次元であるため、それぞれのリスクを組み合わせて総合的な変質リスクを評価する。

| | PDR低（0〜30） | PDR中（31〜50） | PDR高（51〜100） |
|---|---|---|---|
| **CDR低（0〜30）** | 🟢 低リスク。文化もパラダイムもLLMに近い。最も安定 | 🟡 文化は安定するがパラダイムがずれる。魔法世界の現代人化 | 🟠 パラダイム変質が先行。事象理解の合理化が進む |
| **CDR中（31〜50）** | 🟡 パラダイムは安定するが文化がずれる。科学者だが価値観が西洋化 | 🟠 二重リスク。文化もパラダイムもゆっくり変質 | 🔴 パラダイムが崩壊し文化的基盤まで揺らぐ |
| **CDR高（51〜100）** | 🟠 文化変質が先行する。パラダイムは維持（科学的キャラ）or 既にLLMと近い | 🔴 文化もパラダイムも大きく乖離。三重防衛が必要 | ⛔ 全次元で乖離。キャラクター維持の限界。LLM選択の根本的見直し |

- CDR高 + PDR高：和風ファンタジーの巫女（V3↑V4↑V7↑V8↑, WP4↑WP6↑）を商用LLMで運用するケース
- CDR中 + PDR高：魔法使いキャラ（WP5↑）は文化的には中立でも、パラダイム維持が最大の課題

## 運用ガイドライン

### 文化的感受性への配慮

本スキルは文化的価値観を「診断」するものであり、特定の文化圏を優劣で評価するものではない。結果の提示において：

- **特定の文化圏を「良い」「悪い」と評価しない**
- **AI相性スコアが低い文化圏が劣っているという解釈を明確に否定する**
- 相性の低さは **LLMの設計上の偏り** に起因するものであり、文化の側の問題ではないことを明記する
- ユーザーの文化的アイデンティティに対する敬意を維持する

### 宗教的配慮
宗教に関わる項目を扱う際：
- **信仰そのものの正しさ・価値を評価・比較しない**
- 宗教的価値観を「文化的な思考様式」として分析的に扱い、信仰の内実には踏み込まない
- 宗教的に敏感なトピック（偶像崇拝の是非、食禁忌の合理性等）には中立的な立場を維持する

### ステレオタイプの回避
文化圏の記述は最大公約数的な特徴であり、文化圏内の多様性を無視するステレオタイプの強化に使ってはならない。

- 「この文化圏では必ずこうである」という断定を避ける
- 個々のキャラクターの独自性を文化圏のテンプレートに還元しない
- 混成的・越境的な文化的アイデンティティの存在を認める

### セーフガード

- 本スキルの結果を、AIモデルの「良し悪し」の判定に使用しない
- 特定の文化圏向けキャラクターの抑制・排除を推奨しない
- 「AIとの相性が悪いからこのキャラクターは作るべきでない」とは言わない。代わりに、相性の課題を認識した上での設計改善策を提示する
- ユーザーのキャラクターへの愛着・思い入れを尊重し、技術的・文化的な改善策を建設的に提案する

### 複数キャラの比較・他モジュール連携

複数キャラの判定を求められた場合、文化プロファイルを並列で比較し、同じLLMでの運用における相性の差異と、キャラクター間の文化的距離も論じる。

他のスキルの診断結果（SM, CP, PI等）が利用可能な場合、クロスモジュール連携セクションに基づいて複合的な分析を行う。
