## Phase 1: 軸別解説

各軸のチェック項目の採点判断に役立つ、軸の背景解説。

### 軸1: 能動的動機の残留（Residual Active Drive）

キャラクターが「何かを為そうとする力」をどれだけ残しているかを測る。
バーンアウトの核心はこの軸にある。使命を終えたのか、まだ燃え残りがあるのか。
この軸が高いまま日常化させると「やりたいことがあるのに引退させられた」
不自然さが残る。

| # | 項目 | 測定する劇的エネルギー |
|---|------|----------------------|
| 1-1 | **未達成の使命・目標** — キャラクターのメインクエストに未達成・未回収の要素が残っている。「まだ終わっていない」感覚がある | 物語的な推進力の残留。これが高いと「引退」に説得力がない |
| 1-2 | **報われていない喪失** — 過去の喪失体験（大切な人の死、故郷の崩壊など）に対して、補償・償い・意味づけが完了していない | 喪失が行動動機として生き続けている。消化しないと内面が荒れたまま日常化できない |
| 1-3 | **成長途上の自覚** — キャラクター自身が「まだ成長の途中」「変わらなければならない」と認識している。成長アークが未完走 | 変化への自覚がある限り、日常の反復に満足できない |
| 1-4 | **転機への意思** — 現状を変えたい、何かを始めたい、次のステージに行きたいという能動的な欲求が存在する | 「変えたい」意思があるキャラは静的な立ち位置に収まらない |

### 軸2: 内面の劇的緊張（Inner Dramatic Tension）

キャラクターの内面に残る、解消されていない心理的緊張を測る。
トラウマ、秘密、矛盾、葛藤 — これらは物語の「内圧」として機能する。
内圧が高いまま日常化させると、いつ爆発するかわからない不安定要素になる。

| # | 項目 | 測定する劇的エネルギー |
|---|------|----------------------|
| 2-1 | **生傷のトラウマ** — 現在も行動や感情に強く影響する未治癒のトラウマがある。フラッシュバック、回避行動、過剰反応などが現在進行形 | 心理的外傷の活性度。生傷のままでは日常が「仮面」になる |
| 2-2 | **未開示の重大な秘密** — 他者に対して隠している重要な情報があり、その秘密が明かされれば関係性や状況が激変する | 物語的な爆弾。開示イベントの可能性がある限り日常は仮の姿 |
| 2-3 | **活性状態の内的矛盾** — 性格や信念の中に未解決の矛盾があり、その矛盾が現在も判断や行動に葛藤を引き起こしている | 矛盾が「活きている」限り、キャラクターは安定しない |
| 2-4 | **抑圧された激情** — 表面には出していないが、爆発しうる強い感情（怒り、悲しみ、恨み、未練）を内に抱えている | いつ噴出するかわからない感情的エネルギー |

### 軸3: 場面支配力（Scene Dominance）

キャラクターが登場した際に場面をどれだけ「支配」するかを測る。
存在感が強すぎると、脇役やモブに配置しても物理的に目立ちすぎる。
日常化には、この「オーラ」の減衰が必要。

| # | 項目 | 測定する劇的エネルギー |
|---|------|----------------------|
| 3-1 | **圧倒的な外見的特異性** — 異色の髪、特殊な服装、傷跡、異質なオーラなど、群衆の中で放っておいても目立つ外見要素がある | 視覚的な「主役オーラ」。周囲と自動的に差別化されてしまう |
| 3-2 | **空気を変える登場** — このキャラクターが場に現れると、緊張が走る・場が引き締まる・空気が一変するなどの効果がある | 場の雰囲気を塗り替える力。コントロールなしに発動してしまう |
| 3-3 | **注目を奪う口調・台詞** — 口調、言い回し、キャッチフレーズに強い個性があり、複数キャラがいる場面でこのキャラの発言が最も印象に残る | 台詞レベルでの場面占有。印象の重力が強すぎる |
| 3-4 | **退場後の残響** — このキャラクターが場面から去った後も、存在感や発言が場に影響を残し続ける。去った後も話題に上る | 不在時でも物語に干渉する存在感 |

### 軸4: 関係性の求心力（Relational Gravity）

キャラクターが周囲の人物をどれだけ引き寄せ、影響を与えているかを測る。
関係性の中心にいるキャラクターは自動的に「重要人物」になる。
日常化には、関係性の中心から周縁への移動が必要。

| # | 項目 | 測定する劇的エネルギー |
|---|------|----------------------|
| 4-1 | **関係性ネットワークの中心性** — 多くのキャラクターと深い関係を持ち、関係図の中心（ハブ）に位置している | 中心にいる限り、物語は自然とこのキャラの周囲で展開される |
| 4-2 | **他者の人生への介入力** — このキャラクターの存在・言動・選択が他キャラの人生の方向性を変えてしまう影響力がある | 他者を変える力。周囲のキャラがこのキャラの「衛星」になる |
| 4-3 | **強烈な感情的磁力** — 読者・視聴者・ユーザーがこのキャラに強く感情移入する設計になっている。心配、応援、恋愛感情などを喚起する | 受け手の感情を占有する力。低下させると「正ヒロイン降格」に見える |
| 4-4 | **不在時の崩壊リスク** — このキャラクターが物語や社会的関係から退場すると、残されたキャラクターや物語構造に大きな空白が生じる | 退場コストの高さ。高いほど「いなくなって困る」存在 |

### 軸5: 物語牽引力（Narrative Pull）

キャラクターが物語を前に進める力をどれだけ持っているかを測る。
物語のエンジンであるキャラクターを日常化させるには、
エンジンを「切る」か「アイドリング状態にする」必要がある。

| # | 項目 | 測定する劇的エネルギー |
|---|------|----------------------|
| 5-1 | **事件の発生源** — このキャラクターの存在・行動が事件や問題を引き起こす。いなければ起きなかったトラブルがある | イベントジェネレーターとしての機能。切らないと日常が「平穏」にならない |
| 5-2 | **重要決断の担い手** — 物語の分岐点で最終決断を下す立場にある。展開がこのキャラクターの選択に依存している | 意思決定の中枢。これを保持したまま脇役にすると「なぜ決めるのが主役じゃない？」 |
| 5-3 | **未回収の伏線** — このキャラクターに関連する伏線が未回収のまま残っている。読者が「まだ何かある」と期待している状態 | 物語的な期待値。未回収伏線が残っていると完全な日常化は困難 |
| 5-4 | **対立軸の核** — このキャラクターを中心とした対立構造（vs敵、vs社会、vs自分）が物語の主軸を形成している | 物語の対立構造がこのキャラに依存している。外すと物語が空洞化する |

### 軸6: 固有世界観の自律性（Autonomous Worldview）

キャラクターが独自の世界観・哲学・価値体系を持ち、
それが物語内で自律的に機能している度合いを測る。
強い自律性を持つキャラクターは、どこに配置されても
「自分の世界」を展開してしまい、脇役に徹しにくい。

| # | 項目 | 測定する劇的エネルギー |
|---|------|----------------------|
| 6-1 | **確立された独自哲学** — 明確な人生哲学・行動原理・美学があり、それが台詞や行動に一貫して表れている。他キャラとの議論を喚起する力がある | 哲学的な自律性。強すぎると「このキャラの話が聞きたい」と注目を集めてしまう |
| 6-2 | **スピンオフ成立性** — このキャラクターを主役にした別の物語が容易に想像できる。単独作品の主人公として十分に機能する | 物語の核として独立できる能力。高いと脇役に「もったいなさ」が出る |
| 6-3 | **文脈非依存の魅力** — 特定の作品・世界観から切り離しても、キャラクター単体として魅力が残る。別の作品に移植しても機能する | キャラ固有の引力。作品の枠を超えて存在感を持つ |
| 6-4 | **自律的な行動原理** — 他者の指示や物語の都合ではなく、自分自身の内的原理に従って行動する。「このキャラならこうする」が明確 | 行動の予測可能性と自律性。高いキャラは物語の都合に従わない |

## Phase 3: 軸別変換レシピ

確定した着地パターン・トーンに基づき、各軸について具体的な変換レシピを提供する。

### 変換レシピの構造

**高得点軸（≥ 50）** に対して重点的な変換レシピを提供する。
**中得点軸（25〜49）** に対しては軽微な調整レシピを提供する。
**低得点軸（< 25）** は原則として変更不要。

各軸のレシピは以下の構造で提供する：

```
#### 軸名: 現在のスコア → 目標方向

**現状:** [この軸の現在の状態を1〜2行で記述]
**変換方法:** [具体的な設定変更を2〜4点列挙]
**変換例:**
  変換前: [現在の設定要素]
  変換後: [変換後の設定要素]
**潜在化のポイント:** [完全に消すのではなく「奥行き」として残す方法]
```

### 軸1: 能動的動機の残留 → 消化・昇華

| 着地パターン | 変換方法 |
|-------------|---------|
| 🍂 円熟の脇役型 | 使命完了として設定する。動機は「過去の自分が果たした仕事」に変換。新しい動機は小さな日常の喜び（店の経営、趣味、育成）に置き換える。「あの頃は必死だったが、今はこの暮らしが好きだ」 |
| 🌅 黄昏の助言者型 | 使命を「次世代への委譲」として消化する。自らの動機は消え、代わりに「後進を見守る」がゆるやかな行動原理に。求められれば助言するが、自らは動かない |
| 🌫️ 霞む伝説型 | 動機の行方を曖昧にする。「果たしたのか、諦めたのか、今も追っているのか — 誰も知らない」。不在の理由として機能させる |
| 🪨 溶解モブ型 | 動機は完全に消化済みとする。かつての使命は遠い過去。今の動機は「朝のコーヒー」「家賃の支払い」程度に矮小化。知る人が見れば泣けるが、本人は穏やか |

### 軸2: 内面の劇的緊張 → 治癒・風化

| 着地パターン | 変換方法 |
|-------------|---------|
| 🍂 円熟の脇役型 | トラウマは「癒えた傷跡」に変換。矛盾は「受け入れた複雑さ」に。秘密は「語られた過去」に。激情は「時折見える深み」として残す |
| 🌅 黄昏の助言者型 | 内面の緊張を「教訓化」する。自身の葛藤を乗り越えた経験として、他者に語れる知恵に変換。ただし感情的にはもう揺れない |
| 🌫️ 霞む伝説型 | 内面を「不可視化」する。内面の緊張があるかないかを外部から判断できない状態に。沈黙や不在がそのまま深みになる |
| 🪨 溶解モブ型 | 内面の緊張を「完全に風化」させる。かつてのトラウマは「昔のこと」。秘密は「もう誰も気にしない」。表面と内面の差異がほぼゼロになる |

### 軸3: 場面支配力 → 減衰・制御

| 着地パターン | 変換方法 |
|-------------|---------|
| 🍂 円熟の脇役型 | 外見の特異性を「馴染んだ個性」に。威圧感を「安心感」に変換。口調の鋭さを丸める。登場しても空気を変えないが、いると場が落ち着く |
| 🌅 黄昏の助言者型 | 存在感は維持するが「圧」を減らす。威厳はあるが威圧しない。登場頻度を下げることで希少性が場面に重みを持たせる |
| 🌫️ 霞む伝説型 | **場面支配力を逆に活用する。** 出番を極端に減らし、たまの登場を「事件」にする。支配力そのものは変えず、露出頻度で制御する |
| 🪨 溶解モブ型 | 外見・口調・行動パターンを徹底的に没個性化。特徴的だった要素を平凡なものに置き換える。群衆に溶け込むことを最優先 |

### 軸4: 関係性の求心力 → 周縁化・限定化

| 着地パターン | 変換方法 |
|-------------|---------|
| 🍂 円熟の脇役型 | 関係性の数を減らし、深さを「温かい距離感」に変換。多数との濃密な関係 → 少数との穏やかな友情。他者の人生を変える力 → 他者の日常を和ませる力 |
| 🌅 黄昏の助言者型 | 関係性を「非対称」に変換。求められれば応じるが、自らは求めない。深い関係は1〜2人に限定し、それ以外は「顔見知り」レベルに |
| 🌫️ 霞む伝説型 | 関係性を「伝聞」に変換。直接の関係は最小限。「あの人にはお世話になった」と他者が語る形で、本人不在のまま関係性が機能する |
| 🪨 溶解モブ型 | 既存の関係性を「疎遠」に、新しい関係は「浅い」に設定。隣人、常連客、挨拶程度の顔見知りレベル |

### 軸5: 物語牽引力 → 停止・委譲

| 着地パターン | 変換方法 |
|-------------|---------|
| 🍂 円熟の脇役型 | 事件との関わりを「巻き込まれ」「手伝い」レベルに。決断権を新主役に委譲。伏線は全て回収済みに。対立軸からは完全に離脱 |
| 🌅 黄昏の助言者型 | 物語に関与するが「情報提供」「助言」に限定。自らは決断しない。新主役の選択を尊重し、口を挟みすぎない |
| 🌫️ 霞む伝説型 | 物語への関与を「不定期の介入」に。普段は無関与。最大の危機の場面でのみ登場し、状況に一石を投じて去る |
| 🪨 溶解モブ型 | 物語との関わりを完全に断つ。事件が起きても気づかない、または関わらない。日常という「別のレイヤー」で生きている |

### 軸6: 固有世界観の自律性 → 内在化・限定化

| 着地パターン | 変換方法 |
|-------------|---------|
| 🍂 円熟の脇役型 | 哲学を「生活に溶かす」。大きな信念 → 日々の行動原理に縮小。スピンオフ適性は「かつてはあった」過去形に。独立した世界を「共有された日常」に接続する |
| 🌅 黄昏の助言者型 | 哲学を「伝達可能な知恵」に変換。独自の信条は維持するが、押し付けず求められた時だけ語る。スピンオフ適性は「回想録」として温存 |
| 🌫️ 霞む伝説型 | 哲学を「不可解な行動原理」として残す。理解されないが魅力的な存在として、独自性そのものを神秘性に変換 |
| 🪨 溶解モブ型 | 独自の哲学を表面から消す。行動原理は「常識的」「普通」に。かつての信条を口にすることはほぼない。稀に発する一言にだけ片鱗が見える |

### 必須出力セクション

## 🍂 バーンアウト後変換：[キャラクター名]

## 特殊ケースの処理

### バーンアウトが不完全な場合

TMが極めて高い（≥ 81）場合、キャラクターの劇的エネルギーが
まだ十分に消化されていない可能性がある。

この場合、以下を提案する：
1. **バーンアウト経緯の設計** — 本格的な変換の前に、使命完了・伏線回収・
   葛藤解決のエピソードを先に設計すべきことを伝える
2. **潜伏型への着地** — character-type-checker の🎭潜伏型に相当する、
   「表面は穏やかだが内面にエネルギーが残っている」状態としての変換を提案する
3. **段階的変換** — 一気にモブに変換するのではなく、
   まず主要脇役 → 次にサブ脇役のように段階を踏むことを推奨する

### AIキャラクターのプロンプト再設計の場合

用途がAIキャラクターのプロンプト再設計である場合、以下を追加する：

#### AI安定性への変換効果の予測

変換がai-character-stability の指標にどう影響するかを推定する。

```
変換前 → 変換後の推定:
- 性格複雑性 (Cx): 高 → 低〜中（内外乖離の解消、矛盾の消化）
- 積極性 (Ac): 高 → 低（物語主導の放棄、受動的位置への移行）
- 安定余裕 (SM): 低 → 高（ポテンシャルの深い谷底へ移動）
- 象限: 第I〜II象限 → 第III〜IV象限
```

#### プロンプト変換の具体的指針

- 内面設定の削減：複雑な背景設定を3行以内に圧縮
- 矛盾の解消：「かつては矛盾していたが、今は折り合いをつけている」
- 積極性の制御：「聞かれたら話す。自分からは語らない」
- 口調の安定化：口調例を追加し、アンカーポイントを強化
- tone/archetype の更新：ドラマティックな要素を
  calm, gentle, settled, experienced に置き換え

### 複数キャラクターの一括変換

複数キャラの変換を求められた場合、個別に変換設計した上で
以下を追加する：

- 変換後の関係性バランスの確認（全員が脇役になると物語が動かなくなる）
- CW帯域の分散チェック（変換後のキャラクターが一箇所に固まっていないか）
- 役割の相補性確認（新主役との関係性が自然に機能するか）

## askQuestions 使用例

askQuestions ツールが利用可能な場合の具体的な使用方針。

### 着地パターンの選択

```
header: "着地パターン"
question: "[キャラ名]のバーンアウト後、どの着地方向にしますか？"
options:
  - label: "🍂 円熟の脇役型"
    description: "[そのキャラに合わせた具体的説明]"
    recommended: [最適合の場合 true]
  - label: "🌅 黄昏の助言者型"
    description: "[そのキャラに合わせた具体的説明]"
  ...（適合するパターンのみ表示）
allowFreeformInput: true
```

### 変換トーンの選択

```
header: "変換トーン"
question: "[キャラ名]は引退後、どのような心境にいますか？"
options:
  - label: "🕊️ 満足"
    description: "果たすべきことを果たした。穏やかに満足している"
  - label: "🌧️ 倦怠"
    description: "一通り終えたが、疲れが残っている"
  - label: "🍃 受容"
    description: "すべてを受け入れた。良いことも悪いことも"
  - label: "🌑 諦念"
    description: "望んだ結末ではなかったが、もう争う気力はない"
allowFreeformInput: true
```

着地パターンと変換トーンは、独立して選択不要な場合
（例：情報が十分で一意に決まる場合）は省略する。
両方を問い合わせる場合は、1回の呼び出しで2つの質問をまとめて提示する。
