# Character Role Analyzer — キャラクター役割適性判定スキル

## 概要

このスキルは、キャラクターの設定情報を **6つの評価軸・計24項目** で分析し、
物語における **キャラクター重量（CW: Character Weight）** を0〜100で算出する。
CW に加え、各軸のバランスパターンから **役割適性プロファイル** と
**配役ミスマッチ診断** を行い、最適な配役と調整アドバイスを提供する。

### このスキルが解決する問題

創作において、キャラクターの設定と物語上の役割が噛み合わないことは頻繁に起きる。

- 脇役のつもりが主役を食ってしまう「シーンスティーラー」
- 主役なのに印象が薄い「空気主人公」
- モブなのに設定が重すぎて浮いてしまう「過剰設定モブ」
- 深い内面があるのに出番が少なく宝の持ち腐れになる

こうした問題は、キャラクターの持つ「物語的な重さ」と、
割り当てられた「役割の器」のサイズが合っていないときに発生する。
本スキルはこのズレを数値化し、可視化する。

## 判定フロー

```
1. ユーザーからキャラクター情報を受け取る
2. 情報が不十分なら最低限の質問で補完する
3. 6軸×4項目のチェックリストを内部で評価する
4. キャラクター重量（CW）と軸別スコアを算出する
5. 5つの役割に対する適性スコアを算出する
6. 配役ミスマッチ診断パターンをチェックする
7. 結果レポートを出力する（ビジュアル＋考察＋アドバイス）
```

## 情報収集（必要最低限）

判定に最低限必要な情報は以下の4点。ユーザーの入力からこれらが読み取れない場合のみ、
簡潔に質問する。すでに十分な情報があれば質問せずに判定に進む。

1. **キャラクターの外見・第一印象**（例：銀髪の眼帯少女、地味なサラリーマン…）
2. **性格と内面**（例：明るいが実は孤独を抱えている、純朴で裏表がない…）
3. **物語における行動傾向**（例：自ら事件を起こす、巻き込まれる、傍観する…）
4. **他キャラとの関係性**（例：主人公の幼馴染、誰とも深く関わらない…）

**ユーザーが意図している役割（「モブのつもり」「主役として作った」など）** が
提示されている場合は、その意図と判定結果のズレを重点的に分析する。

作品名やキャラ名が提示された既存キャラの場合は、自身の知識から補完してよい。

## 評価チェックリスト（6軸 × 4項目 = 24項目）

各項目を以下の3段階で採点する：

| 記号 | 意味 | 点数 |
|------|------|------|
| ◯ | 明確に該当する | 2 |
| △ | 部分的に該当 / 状況による | 1 |
| ✕ | 該当しない | 0 |

---

### 軸1: 外見・印象の特異性（Distinctiveness）

キャラクターが読者・視聴者の記憶に残るかどうかの第一関門。
モブは群衆に溶け込み、主役は一目で「こいつが主人公だ」とわかる。
この軸が高いキャラクターは、登場するだけで視線を集める。

| # | 項目 | 説明 |
|---|------|------|
| 1-1 | **視覚的特異性** | 外見に目立つ特徴がある（異色の髪、傷跡、特殊な服装、奇抜な体格など）。群衆の中で一目で見つけられるか？ |
| 1-2 | **名前・呼称のインパクト** | 名前、異名、二つ名、キャッチフレーズなどが印象的で記憶に残る。「あのキャラ」と指し示しやすいか？ |
| 1-3 | **初登場のインパクト** | 初登場シーンが印象に残る演出・行動・台詞を伴う設計になっている。強烈な第一印象を与えるか？ |
| 1-4 | **アイコン性** | そのキャラクターを象徴するアイテム、ポーズ、口癖、行動パターンがある。シルエットだけで誰かわかるか？ |

---

### 軸2: 物語推進力（Narrative Drive）

キャラクターが物語を「前に進める力」を持っているかどうか。
主役は物語のエンジンであり、モブは物語の背景である。
この軸が高いキャラクターは、いるだけでストーリーが動き出す。

| # | 項目 | 説明 |
|---|------|------|
| 2-1 | **能動的な目標** | 自分自身の明確な目標・欲求・使命を持ち、それに向かって行動する。物語の方向性を自ら決めるか？ |
| 2-2 | **葛藤・対立の生成** | その存在が他キャラや社会との対立・葛藤を自然に生み出す。このキャラがいなければ起きない問題があるか？ |
| 2-3 | **決断の主体性** | 物語の分岐点で決断を下す立場にある。重要な選択を迫られ、その結果が展開に影響するか？ |
| 2-4 | **成長アーク** | 物語を通じて内面的な変化・成長・堕落が設計されている。始まりと終わりで別人になるか？ |

---

### 軸3: 内面の深度（Inner Depth）

キャラクターの心理的な奥行き。層が深いほど「掘れば掘るほど味が出る」。
主役や重要脇役には必要だが、モブに持たせると設定が宝の持ち腐れになる。

| # | 項目 | 説明 |
|---|------|------|
| 3-1 | **多面的な性格** | 表面的な性格の裏に別の一面がある（優しいが冷酷な判断もできる、陽気だが深い孤独を抱えるなど）。一言では説明しきれないか？ |
| 3-2 | **過去の重み** | 現在の行動や性格に強く影響するバックストーリーがある。過去を知ると見え方が変わるか？ |
| 3-3 | **内的矛盾** | 性格や信念の中に自覚的・無自覚的な矛盾がある。その矛盾がキャラクターの魅力になっているか？ |
| 3-4 | **秘密・未開示情報** | 読者や他キャラにまだ明かされていない重要な情報を持っている。「実は…」と開示する余地があるか？ |

---

### 軸4: 関係性の磁力（Relational Magnetism）

他のキャラクターとの結びつきの強さと密度。
主要キャラは関係性の結節点であり、モブは関係性の外縁にいる。
この軸が高いキャラクターは、周囲を巻き込んで人間ドラマを生む。

| # | 項目 | 説明 |
|---|------|------|
| 4-1 | **関係性の数と多様性** | 複数のキャラと異なるタイプの関係（友情、敵対、師弟、恋愛など）を持っている。関係図の中心に近いか？ |
| 4-2 | **他者への影響力** | このキャラの言動が他キャラの感情・行動・価値観に変化を与える。他キャラの人生を変える力があるか？ |
| 4-3 | **感情的な求心力** | 読者・視聴者がこのキャラに感情移入しやすい設計になっている。応援したい、心配になる、気になるか？ |
| 4-4 | **不在の影響** | このキャラが退場・不在になった場合、物語や他キャラに大きな影響が出る。いなくなったら話が成り立たないか？ |

---

### 軸5: 場面占有力（Scene Presence）

キャラクターが登場する場面をどれだけ「支配」するか。
これが高いキャラクターは、脇役やモブとして配置しても目立ちすぎる。
逆に低いと主役でも存在感が薄い。配役ミスマッチの主要因。

| # | 項目 | 説明 |
|---|------|------|
| 5-1 | **台詞の引力** | 口調、言い回し、声のトーンに独自性があり、台詞を読むだけで誰の発言かわかる。台詞に個性が出ているか？ |
| 5-2 | **空気の変容** | このキャラが登場すると場の雰囲気が変わる（緊張が走る、和む、不穏になるなど）。場の空気を塗り替えるか？ |
| 5-3 | **注目の奪取** | 複数キャラがいる場面で、読者の注目がこのキャラに集まりやすい。他キャラを「背景化」させてしまうか？ |
| 5-4 | **退場後の残響** | このキャラが場面から退出した後も、その存在感や発言が場に影響を残す。去った後も忘れられないか？ |

---

### 軸6: 独立性（Independence）

キャラクターが他のキャラや特定の場面に依存せず、
単独で物語を牽引・維持できる力。主役には必須だが、
モブや脇役には過剰になりうる。スピンオフ適性とも言える。

| # | 項目 | 説明 |
|---|------|------|
| 6-1 | **単独での成立性** | このキャラだけで1つのエピソードやシーンが成立する。独り語りや単独行動のシーンが成り立つか？ |
| 6-2 | **固有の世界観** | このキャラ独自の価値観・哲学・美学が確立されている。「この人の世界」と呼べるものがあるか？ |
| 6-3 | **スピンオフ適性** | このキャラを主役にした別の物語が想像できる。単独作品の主人公として成立するか？ |
| 6-4 | **文脈非依存性** | 特定の作品・世界観から切り離しても、キャラクターとして魅力が残る。別の作品に移植しても機能するか？ |

## スコア算出方法

### 1. 軸別スコアの集計

各軸のスコアを算出する（各軸4項目×最大2点 = 最大8点 → 100点満点に正規化）。

```
軸スコア = round(軸の生スコア / 8 × 100)
```

6軸のスコアをそれぞれ算出する：
- D1: 外見・印象の特異性
- D2: 物語推進力
- D3: 内面の深度
- D4: 関係性の磁力
- D5: 場面占有力
- D6: 独立性

### 2. キャラクター重量（CW: Character Weight）の算出

全24項目の合計から算出する。CWはキャラクターの「物語的な重さ」を示す総合指標。

```
CW生スコア = 全24項目の合計（最大48）
CW = round(CW生スコア / 48 × 100)
```

### 3. 役割適性スコアの算出

各役割には「理想的な軸バランス」がある。
各役割の理想プロファイルとの一致度を適性スコアとして算出する。

#### 理想プロファイル（各軸の理想範囲）

**主役（Protagonist）**の理想像：
すべての軸が高水準。特に物語推進力と独立性が高い。

```
理想: D1≥75, D2≥75, D3≥63, D4≥63, D5≥63, D6≥75
重み: D1=1.0, D2=1.5, D3=1.0, D4=1.0, D5=1.0, D6=1.5
```

**主要脇役（Major Supporting）**の理想像：
存在感はあるが物語の主導権は主役に譲る。関係性の磁力が高い。

```
理想: D1=50-75, D2=38-63, D3=50-75, D4≥63, D5=38-63, D6=25-50
重み: D1=1.0, D2=0.8, D3=1.0, D4=1.5, D5=1.0, D6=0.8
```

**サブ脇役（Minor Supporting）**の理想像：
一つ二つの特徴で印象を残す。出番は限られるが、いると場が豊かになる。

```
理想: D1=25-63, D2=13-38, D3=25-50, D4=25-50, D5=25-50, D6=13-38
重み: D1=1.2, D2=0.8, D3=0.8, D4=1.0, D5=1.2, D6=0.8
```

**ワンタイム（One-time Appearance）**の理想像：
一度の登場で強烈な印象を残す。深い内面があっても開示は限定的。
場面占有力が高いが、関係性は限定的。

```
理想: D1≥63, D2=25-50, D3=25-63, D4=13-38, D5≥63, D6=38-63
重み: D1=1.5, D2=0.8, D3=0.8, D4=0.5, D5=1.5, D6=1.0
```

**モブ（Background）**の理想像：
群衆の一部として機能する。個性は最小限で、場面の背景に溶け込む。

```
理想: D1≤25, D2≤13, D3≤25, D4≤25, D5≤25, D6≤13
重み: D1=1.0, D2=1.0, D3=1.0, D4=1.0, D5=1.5, D6=1.0
```

#### 適性スコアの計算ロジック

各役割の適性スコアは以下のように計算する：

```
1. 各軸について、キャラのスコアと理想範囲の「距離」を求める
   - 理想範囲内 → 距離 = 0（完全一致）
   - 理想範囲外 → 距離 = 範囲の最寄り端からの差分
2. 距離に重みを掛けて合算し、ペナルティ値とする
3. 適性スコア = max(0, 100 - ペナルティ値)
```

具体的な計算：

```
penalty = 0
for each axis i:
    if score[i] < ideal_min[i]:
        gap = ideal_min[i] - score[i]
    elif score[i] > ideal_max[i]:
        gap = score[i] - ideal_max[i]
    else:
        gap = 0
    penalty += gap × weight[i] × 0.5

fitness = max(0, round(100 - penalty))
```

乗数0.5は、1軸の最大ギャップ（100点）が過剰にペナルティを与えないための調整。

## 配役ミスマッチ診断パターン

CWと軸バランスの組み合わせから、以下の診断パターンを検出する。
キャラクターの意図された役割（ユーザー申告）と照合して警告を出す。

### 🎭 シーンスティーラー（Scene Stealer）

**条件:** D5（場面占有力）≥ 63 かつ D2（物語推進力）< 50
**意味:** 自分でストーリーを動かすわけではないのに、登場するたびに場を支配してしまう。
脇役やモブとして配置すると主役を食う危険がある。
**典型例:** 強烈な口癖や外見を持つ「賑やかし」キャラ、圧倒的カリスマの先輩キャラ。

### 👻 空気主人公（Invisible Protagonist）

**条件:** D2（物語推進力）≥ 63 かつ D5（場面占有力）< 38
**意味:** 物語を進める役割は果たしているが、場面上の存在感が薄い。
読者に「この人が主役だったの？」と言われるリスクがある。
**典型例:** 巻き込まれ型主人公で口調に個性がないキャラ。

### 📦 設定過剰モブ（Over-designed Background）

**条件:** CW ≥ 40 だが意図された役割がモブ
（または D3 ≥ 50 かつ D5 < 25）
**意味:** モブとしては設定が重すぎる。登場するたびに「このキャラ掘り下げないの？」
と読者が感じ、期待を裏切ることになる。昇格させるか設定を削るべき。
**典型例:** 重い過去を持つ名もなき店主、実は元勇者の村人A。

### 🌑 埋没する才能（Buried Potential）

**条件:** D3（内面の深度）≥ 63 かつ D1（外見・印象）< 38 かつ D5（場面占有力）< 38
**意味:** 内面は深く魅力的だが、外見や場面での印象が薄いため、
読者がその魅力に気付く前に素通りされてしまう。
**典型例:** 地味な見た目で深い哲学を持つキャラ、壮絶な過去を持つ無口キャラ。

### 🌀 ブラックホール（Narrative Black Hole）

**条件:** D4（関係性の磁力）≥ 75 かつ D5（場面占有力）≥ 75
**意味:** 周囲のキャラクターや場面を全て吸い込む圧倒的な存在感。
主役にすれば作品が成立するが、脇役に置くと他の全キャラが衛星化する。
群像劇には向かない。
**典型例:** 絶対的カリスマの王、魅力的すぎるヒロイン/ヒーロー。

### 🎪 一発屋の逸材（Born for One Scene）

**条件:** D1（外見・印象）≥ 75 かつ D6（独立性）≥ 50 かつ D4（関係性の磁力）< 38
**意味:** 強烈な第一印象と独立した世界観を持つが、他者と関係を結びにくい。
ワンタイムのゲスト登場で最大効果を発揮するが、
レギュラーにすると孤立して場が持たない。
**典型例:** 通りすがりの達人、一話完結のライバル。

### 🪞 万能カメレオン（Universal Adapter）

**条件:** 全軸が25〜63の範囲に収まっている
**意味:** 突出した特徴がない代わりに、どの役割にもそれなりに適応できる。
扱いやすいが、印象に残りにくい。主役にするなら何か一つ尖らせるべき。
**典型例:** バランス型の常識人キャラ、「普通の高校生」。

### ⚡ 不安定な核弾頭（Unstable Warhead）

**条件:** D2（物語推進力）≥ 75 かつ D3（内面の深度）≥ 75 かつ D5（場面占有力）≥ 75
**意味:** 物語を動かす力、内面の深さ、場面の支配力すべてが極めて高い。
主役としては最高だが、脇役に置くと制御不能。
話のどこに置いても物語がこのキャラ中心に回り始める。
**典型例:** 作者の「推し」が入りすぎたサブキャラ。

## CW（キャラクター重量）による目安分類

CWは「このキャラクターが物語でどれくらいの場所を占めるか」の総合指標。
意図した役割のCW帯域とのズレが大きいほど、配役ミスマッチのリスクが高い。

| CW | 自然な立ち位置 | 説明 |
|----|---------------|------|
| 85〜100 | 🌟 **主役級** | 物語の中心になる設計。脇役に置くと他を圧倒する危険あり |
| 65〜84 | 🔶 **主要脇役級** | 十分な存在感と深みがある。主役にもなれるが、脇役として最も輝く帯域 |
| 40〜64 | 🔹 **サブ脇役級** | 適度な個性と役割がある。レギュラーだが出番は限定的が自然 |
| 20〜39 | 🔸 **ワンタイム級** | 一場面で印象を残す程度の設定。継続登場させるなら設定の追加が必要 |
| 0〜19 | ⬜ **モブ級** | 群衆の一部として機能する。個性は最小限。背景として最適 |

## 配役アドバイスの出力ガイドライン

### アドバイスの構成

1. **現在の設計に対する評価**（2〜3文）
   - CWと軸バランスから見た「このキャラの自然な立ち位置」を伝える

2. **もし [別の役割] にするなら**（役割ごとに2〜3文ずつ）
   - ユーザーが意図した役割、または適性スコアが低い役割に合わせるために
     「何を足すべきか」「何を削るべきか」を具体的に提案する
   - 抽象的な「もっと個性を出す」ではなく、
     「口癖を追加する」「過去の設定を削る」レベルの具体性を心がける

### 役割変更アドバイスのテンプレート

**主役に昇格させるなら：**
- D2が低い → 「自発的な目標を設定する。巻き込まれるだけでなく、自ら選ぶ瞬間を作る」
- D5が低い → 「口調に癖をつける、象徴的な台詞を作る。登場時の演出を強化する」

**脇役に降格させるなら：**
- D2が高すぎる → 「目標のスケールを小さくする。物語全体ではなく、1つのサブプロットに限定する」
- D5が高すぎる → 「口調の癖を控えめにする。登場シーンを減らして『ここぞ』の場面に集中させる」

## 運用ガイドライン

### 複数キャラの比較

ユーザーが複数キャラの判定を求めた場合、個別に判定した上で
以下を追加する：

- CWの比較一覧表（キャラ名とCWを並べ、CW帯域を表示）
- 同じ作品内でのバランス分析（CW帯域の偏りがないか、
  主役級が複数いないか、全員がサブ脇役級に固まっていないかなど）
- 軸の相補性（主役が独立性重視なら、脇役は関係性重視が望ましい、など）

### 曖昧な情報への対応

キャラクター情報が曖昧な場合、△（部分的に該当）を積極的に活用する。
無理に◯か✕に振り分けず、判定結果にも「情報が限定的なため
推定値である」旨を添える。

### 既存作品のキャラクターの場合

既存作品のキャラを判定した場合、実際の役割と判定結果のズレを
積極的に分析する。例えば：

- 主役なのにCWが脇役級 → 「周囲のキャラが強すぎる？ 物語構造で補っている？」
- 脇役なのにCWが主役級 → 「作品の真の主役はこちらでは？ スピンオフの余地は？」

こうしたズレは作品の特徴を語る貴重な材料になる。

### 判定の限界の認識

このチェックリストはキャラクターの「設定上のポテンシャル」を可視化するツールであり、
実際の作品での機能は、作者の筆力、物語構造、出番の配分、演出によっても左右される。
CWが低くても演出次第で印象に残ることはあるし、CWが高くても活かしきれないこともある。
判定結果はあくまで「設定だけ見たときの傾向分析」というスタンスで伝える。

### チェックリスト詳細の出力

ユーザーから求められた場合、または判定根拠の説明に必要な場合に、
6軸すべての項目と採点を表形式で提示する。
各項目の採点理由を1行程度の補足で添えると、判定の透明性が高まる。
