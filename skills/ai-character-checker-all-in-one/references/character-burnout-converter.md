# Character Burnout Converter
# — ドラマ型キャラクター バーンアウト後変換スキル

## 概要

このスキルは、**主役級の物語的役割を終えたドラマ型キャラクター**（＝バーンアウト後）を
低エネルギーの日常化・潜在化キャラクターに変換するための体系的な手法を提供する。

### バーンアウトとは何か

物語におけるバーンアウトとは、キャラクターが担っていた劇的機能
（使命の遂行、成長アークの完走、葛藤の解決、伏線の回収など）が
一通り消化された状態を指す。

バーンアウト後のキャラクターには2つの道がある：

1. **退場** — 物語から完全に去る
2. **潜在化・日常化** — 物語に残るが、劇的役割を降ろして存在し続ける

**本スキルは後者を設計する。**

### このスキルが解決する問題

創作においてもAIキャラクター設計においても、以下の問題が頻繁に発生する：

- **主役の燃え尽き** — メインアークを終えた主役が、続編・後日談でどう振る舞えばいいかわからない
- **降格の不自然さ** — 主役から脇役に「格下げ」すると唐突に弱体化して見える
- **残留ドラマの処理** — 使命は終えたが、キャラの劇的オーラが残っており日常系に溶け込めない
- **再配置の設計** — 群像劇で世代交代する際、旧主役をどこに置くべきか判断がつかない
- **AIキャラの安定化** — ドラマ性が高くAI運用で不安定なキャラを安定化したい

これらの問題の本質は、**キャラクターに蓄積された劇的エネルギーを
どう消化・変換して低エネルギー状態に着地させるか**という設計課題である。

### 関連する理論的概念

本スキルは以下の概念を理論的基盤として活用する。

#### ポテンシャルカーブ上の位置移動（ai-character-stability）

ドラマ型キャラクターは**浅いポテンシャルの窪み**（準安定状態）にいる。
劇的エネルギーが高いほど窪みは浅く、外乱で容易にポテンシャル地形上を移動する。

バーンアウト後変換とは、このキャラクターを**より深いポテンシャルの谷底**
（安定平衡点）に移動させる作業に相当する。

```
変換前:              変換後:
    ╲  ╱  ╲           ╲        ╱
     ╲╱    ╲   →       ╲      ╱
      ●     ╲           ╲  ● ╱
   （浅い窪み）          （深い谷底）
```

重要なのは、**谷底への移動は「劣化」ではなく「安定化」である**ということ。
ステレオタイプに堕落するのではなく、劇的エネルギーが消化された結果として
自然に到達する安定状態を設計する。

#### N/D スコアの遷移（character-type-checker）

バーンアウト後変換は、character-type-checker の概念で言えば
**Dスコアの要素をNスコアの要素に変換する**（またはDスコアを消化して無効化する）操作である。

| 変換前（Dスコア要素） | 変換後（Nスコア要素への遷移） |
|-----------------------|------------------------------|
| 使命・宿命がある | 使命は果たした。今は穏やかに暮らしている |
| 喪失体験が行動の根底 | 喪失は癒えた。その経験は静かな深みになった |
| 激情が爆発する | 感情の波は穏やかになった。たまに昔の片鱗が見える |
| 対立を誘発する | 場の空気を和ませる存在になった |
| トラウマが行動に影響 | 傷跡は残るが、もう痛まない |
| 成長しなければ存立しない | 今の自分に満足している |

ただし、**D要素を完全にゼロにするのではなく「潜在化」させる**のが
本スキルの核心である。完全にDを消すとキャラクターの深みが失われる。
D要素を「過去形にする」ことで、日常の裏に透ける奥行きとして機能させる。

#### CW（キャラクター重量）の意図的低下（character-role-analyzer）

バーンアウト後変換は、character-role-analyzer の概念で言えば
**CWを意図的に下げて、より低い役割帯域に着地させる**操作である。

| 変換前CW帯域 | 変換後CW帯域 | 変化幅 |
|-------------|-------------|--------|
| 85〜100 主役級 | 65〜84 主要脇役級 | 小 |
| 85〜100 主役級 | 40〜64 サブ脇役級 | 中 |
| 85〜100 主役級 | 20〜39 ワンタイム級 | 大 |
| 85〜100 主役級 | 0〜19 モブ級 | 最大 |

変化幅が大きいほど多くの劇的要素を消化・変換する必要があり、
変換の難度も上がる。

#### AI安定性への効果

ドラマ型キャラクターのバーンアウト後変換は、AI運用の観点では：

- **性格複雑性（Cx）の低下** — 内面の矛盾・緊張が解消される
- **積極性（Ac）の低下** — 物語を主導する必要がなくなる
- **安定余裕（SM）の向上** — 第IV象限（C高・P高）に近づく

AIキャラクターのプロンプト再設計にも直接活用できる。

## 変換フロー

```
Phase 1: バーンアウト前診断
  1. ユーザーからキャラクター情報を受け取る
  2. 情報が不十分なら最低限の質問で補完する
  3. 残留ドラマエネルギー（6軸×4項目）を評価する
  4. 変換規模（TM）を算出する

Phase 2: 着地方向の決定
  5. TMと軸別スコアから適合する着地パターンを判定する
  6. 複数パターンが適合する場合、ユーザーに選択を求める★
  7. 着地パターンと目標役割を確定する

Phase 3: 変換レシピの生成
  8. 確定した着地パターンに基づき、軸ごとの変換レシピを生成する
  9. 変換前→変換後の設定対照表を出力する
  10. 変換後のキャラクター概要を出力する
```

**★ ユーザーへの問い合わせ:**
方向性が複数考えられる場合は、必ずユーザーに選択を求める。
利用可能であれば **askQuestions** ツールの選択機能
（options付きの質問）を使用し、各選択肢の説明を添える。
利用不可能な場合は、番号付きで候補を提示しユーザーの回答を待つ。
ユーザーが明示的に方向性を指定している場合は問い合わせせずに進む。

## 情報収集

判定に最低限必要な情報は以下の通り。ユーザーの入力から読み取れない場合のみ質問する。

1. **キャラクターの現在の設定**（性格、外見、能力、口調など）
2. **キャラクターが担っていた劇的役割**（主役、どのような物語の主人公だったか）
3. **バーンアウトの経緯**（使命を果たした、敗北した、諦めた、時間が経った…。任意だが有用）
4. **意図する変換後の役割**（主要脇役、モブなど。任意。指定があれば着地方向の選択をスキップ可能）
5. **変換後の用途**（創作用、AIプロンプト再設計、後日談設定など。任意）

作品名やキャラ名が提示された既存キャラの場合は、自身の知識から補完してよい。

## Phase 1: バーンアウト前診断

### 残留ドラマエネルギー評価チェックリスト（6軸 × 4項目 = 24項目）

キャラクターに現在蓄積されている劇的エネルギーを測定する。
**スコアが高いほど、変換で消化すべきドラマエネルギーが大きい。**

各項目を以下の3段階で採点する：

| 記号 | 意味 | 点数 |
|------|------|------|
| ◯ | 明確に該当する（強い劇的エネルギーがある） | 2 |
| △ | 部分的に該当 / 一部消化済み | 1 |
| ✕ | 該当しない（劇的エネルギーなし、または消化済み） | 0 |

> **📎 各軸の詳細な解説（軸の背景説明・各項目の測定対象の詳細）は
> `references/character-burnout-converter--conversion-details.md` § Phase 1: 軸別解説 を参照。**
> 以下の表の「測定する劇的エネルギー」列は、採点判断の要約である。

## スコア算出方法

### 1. 軸別スコアの集計

各軸のスコアを算出する（各軸4項目×最大2点 = 最大8点 → 100点満点に正規化）。

```
軸スコア = round(軸の生スコア / 8 × 100)
```

6軸のスコアをそれぞれ算出する：

- R1: 能動的動機の残留（Residual Active Drive）
- R2: 内面の劇的緊張（Inner Dramatic Tension）
- R3: 場面支配力（Scene Dominance）
- R4: 関係性の求心力（Relational Gravity）
- R5: 物語牽引力（Narrative Pull）
- R6: 固有世界観の自律性（Autonomous Worldview）

### 2. 変換規模（TM: Transformation Magnitude）の算出

全24項目の合計から算出する。TMは「どれだけ大きな変換が必要か」を示す総合指標。

```
TM生スコア = 全24項目の合計（最大48）
TM = round(TM生スコア / 48 × 100)
```

### 3. TM の解釈目安

| TM | 変換規模 | 解釈 |
|----|---------|------|
| 0〜20 | 🟢 **軽微** | すでにかなり消化済み。微調整で日常化可能。自然な降格が容易 |
| 21〜40 | 🟡 **中程度** | 主要な劇的要素の消化が必要。丁寧な変換設計で自然に着地可能 |
| 41〜60 | 🟠 **大規模** | 強い劇的オーラが残っている。複数軸の本格的な変換が必要 |
| 61〜80 | 🔴 **極大** | 物語の中心にいるキャラクター。バーンアウト経緯の設計から必要 |
| 81〜100 | ⛔ **限界的** | 劇的エネルギーが飽和している。完全な日常化は困難。潜伏型（🎭）への着地を推奨 |

### 4. 変換プロファイルの分析

6軸のスコアパターンから、変換の重点領域を特定する。

**高得点軸（≥ 50）** は変換の主要ターゲットであり、
重点的な解消・変換レシピが必要。

**低得点軸（< 25）** はすでに消化済みまたは元々低い領域であり、
大きな変換は不要。

**最高得点軸** はキャラクターの「最も強い劇的個性」であり、
完全に消すとキャラクターの魅力が失われるリスクがある。
「消す」のではなく「潜在化させる」ことが望ましい。

## Phase 2: 着地方向の決定

### 4つの着地パターン

TM・軸別スコア・キャラクターの特性に応じて、以下の4つの着地パターンから
最適なもの（通常は複数候補）をユーザーに提示する。

#### 🍂 円熟の脇役型（Matured Supporting）

**変換先:** 主要脇役〜サブ脇役 / **トーン:** 穏やか、温かい、安定
劇的エネルギーを「穏やかな深み」に変換。かつての激しさは落ち着き、
経験に裏打ちされた安定感のある存在に。物語にはまだ関わるが主導権は握らない。
**向くキャラ:** 使命を果たし終えた元英雄、成長アーク完走済み、平和を手に入れた人物

#### 🌅 黄昏の助言者型（Twilight Mentor）

**変換先:** 主要脇役 / **トーン:** 達観、静かな見守り、鋭さ
劇的エネルギーを「次世代への知恵」に変換。自らは動かないが経験と洞察で
周囲を導く。かつてのドラマは「教訓」として語られ、新主役の鏡になる。
**向くキャラ:** R6（世界観自律性）が高い、影響力が強く背景に溶け込めない人物

#### 🌫️ 霞む伝説型（Fading Legend）

**変換先:** ワンタイム〜サブ脇役 / **トーン:** 不在の存在感、伝聞、希少な登場
劇的エネルギーを「伝聞と不在」に変換。普段は姿を見せず名前だけが語られる。
「いない」ことが存在感になる逆説的な配置。
**向くキャラ:** R3（場面支配力）が極めて高い、伝説化しやすい経歴を持つ人物

#### 🪨 溶解モブ型（Dissolved Background）

**変換先:** モブ / **トーン:** 平凡、静謐、隠された深み
劇的エネルギーを「知る人ぞ知る過去」に変換。外見も口調も平凡にならし
群衆に完全に溶け込ませる。過去を知る者が見れば深みが見える構造。
**向くキャラ:** 意図的に姿を消したい人物、TM低（0〜40）の人物、遅延開示用

### 着地パターンの適合判定

| パターン | 適合条件 |
|---------|---------|
| 🍂 円熟の脇役型 | TM ≤ 60 かつ R1 < 50（動機がある程度消化済み） |
| 🌅 黄昏の助言者型 | R6 ≥ 50（固有世界観が強い）かつ R4 ≥ 38（関係性の影響力が残存） |
| 🌫️ 霞む伝説型 | R3 ≥ 50（場面支配力が高い）または TM ≥ 50 |
| 🪨 溶解モブ型 | TM ≤ 40 または ユーザーが明示的にモブを指定 |

**複数パターンが適合する場合**、ユーザーに選択を求める。
askQuestions ツールが利用可能ならoptions付き選択質問で提示（最適合に recommended 付与）。
利用不可なら番号付きリストで候補を提示し、各候補の具体的イメージを2〜3行添える。

### 変換トーン（Settlement Tone）

バーンアウト経緯やキャラクター性格から複数考えられる場合に確認する。

| トーン | 説明 |
|--------|------|
| 🕊️ 満足（Contentment） | 果たすべきことを果たした。今は穏やかに満足している |
| 🌧️ 倦怠（Weariness） | 一通り終えたが、疲れが残っている。まだどこか重い |
| 🍃 受容（Acceptance） | すべてを受け入れた。良いことも悪いことも、過去の一部 |
| 🌑 諦念（Resignation） | 望んだ結末ではなかったが、もう争う気力はない |

一意に決まる場合、またはユーザーの希望が明確な場合は問い合わせない。

## Phase 3: 変換レシピの生成

確定した着地パターン・トーンに基づき、各軸について具体的な変換レシピを提供する。

**高得点軸（≥ 50）** → 重点的な変換レシピ /
**中得点軸（25〜49）** → 軽微な調整 /
**低得点軸（< 25）** → 変更不要

各軸のレシピは以下の構造で提供する：

```
#### 軸名: 現在のスコア → 目標方向

**現状:** [1〜2行]
**変換方法:** [具体的な設定変更 2〜4点]
**変換例:**
  変換前: [現在の設定要素]
  変換後: [変換後の設定要素]
**潜在化のポイント:** [完全に消すのではなく「奥行き」として残す方法]
```

> **📎 6軸×4パターンの変換ガイドライン全表は
> `references/character-burnout-converter--conversion-details.md` § Phase 3: 軸別変換レシピ を参照。**
> 変換レシピ生成時、高得点軸の該当パターンを参照して具体的なレシピを組み立てる。

出力は以下のセクション構成で行う：

1. **Phase 1: 残留ドラマエネルギー診断** — TM（スコアバー+解釈）、軸別スコア（スコアバー）、診断サマリー（3〜5行散文）
2. **Phase 2: 着地方向** — 適合パターン・選択結果、変換トーン
3. **Phase 3: 変換レシピ** — 高得点軸から順に変換レシピ提示
4. **変換前→変換後 対照表** — 役割/性格/動機/関係性/口調/外見
5. **変換後キャラクター概要** — 散文3〜8行。「かつて何かがあった」と感じられる文章
6. **💡 運用上の注意** — 2〜3点

スコアバー: 20文字幅（█ × 塗りつぶし + ░ × 残り）。1文字 = 5ポイント。

## 特殊ケースの処理

> **📎 各特殊ケースの詳細な処理手順は
> `references/character-burnout-converter--conversion-details.md` § 特殊ケースの処理 を参照。**

以下の特殊ケースに該当する場合、追加の処理を行う：

- **不完全バーンアウト（TM ≥ 81）**: バーンアウト経緯の設計提案、潜伏型着地、段階的変換の推奨
- **AIキャラクターのプロンプト再設計**: AI安定性（SM）への変換効果予測、プロンプト変換指針（内面削減、矛盾解消、積極性制御、口調安定化、tone更新）
- **複数キャラクター一括変換**: 関係性バランス確認、CW帯域分散チェック、役割相補性確認

## 運用ガイドライン

### 変換は「劣化」ではない

本スキルの最重要原則。バーンアウト後変換は**キャラクターの格下げ**ではなく、
**物語的役割の遷移設計**である。

- 円熟した脇役は、駆け出しの主役にはない深みを持つ
- 霞む伝説は、不在そのものが物語に奥行きを与える
- 溶解したモブの「実は…」は、最も強力な遅延開示になりうる

変換後のキャラクターに対して「格下げされた」「弱体化した」という
印象を与えない記述を心がける。「変わった」のではなく
「次の章に進んだ」というニュアンスで伝える。

### 「潜在化」の原則

劇的要素を完全に消去するのではなく、**日常の裏に透ける奥行きとして保存する**。

- ❌ 「使命はなくなった」 → ◯ 「使命は果たし終えた」
- ❌ 「トラウマは消えた」 → ◯ 「傷は癒えた。跡は残っている」
- ❌ 「カリスマ性を削除」 → ◯ 「威厳は穏やかさに形を変えた」
- ❌ 「過去を削除」 → ◯ 「過去は、時折見せる遠い目に宿っている」

### 方向性の問い合わせの判断基準

**問い合わせが必要な場合:**
適合パターンが2つ以上 / トーンが2つ以上考えられる /
バーンアウト経緯が不明で複数解釈可能 / 「核」の判断が難しい

**問い合わせ不要な場合:**
ユーザーが具体的着地点を明示 / 適合パターンが1つのみ / 経緯が明確

> **📎 askQuestions ツールの具体的な使用例（着地パターン選択・変換トーン選択のフォーマット）は
> `references/character-burnout-converter--conversion-details.md` § askQuestions 使用例 を参照。**

### 他スキルとの併用

- **character-type-checker** — 変換前のN/D/Gスコアで現状把握。変換後はN寄りまたは🎭潜伏型が理想
- **character-role-analyzer** — 変換前後のCWと役割帯域の照合。配役ミスマッチ解消の検証
- **ai-character-stability** — AIキャラ再設計時、変換前後のSM変化予測と確認
- **ai-self-description-analyzer** — 蒸留結果への変換適用時、異常スコア（PI）低下予測
- **ai-user-conflict-predictor** — 衝突パターンの変化予測。ドラマ型→日常型でF6低下

### 判定の限界の認識

- 設定文上の変換ガイドラインであり、作品内での機能は作者の筆力に依存する
- 同じレシピでも作品のジャンル・トーン・他キャラとの相性で効果は変わる
- AIキャラの場合、モデル・温度・会話履歴にも挙動は依存する
- 「正解」は1つではなく、複数の着地方向がそれぞれ異なる魅力を持ちうる
